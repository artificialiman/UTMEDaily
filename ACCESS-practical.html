School Database — Student Records Management
The database SchoolDB contains a table named Students with student records for JSS 3. Using the database environment provided, carry out the following tasks carefully.
Phase 1 — Table Design
i. ○ Open the Students table in Design View. Set StudentID as the Primary Key. (3 marks)
ii. ○ Set the data type of DateOfBirth to Date/Time and Score to Number. (3 marks)
iii. ○ Add a new field Grade with data type Short Text and field size 2. (4 marks)
Phase 2 — Data Entry
iv. ○ Switch to Datasheet View and enter at least 8 student records. Ensure no StudentID is repeated. (4 marks)
v. ○ Fill in the Grade field for each student: A (≥70), B (≥60), C (≥50), F (<50). (3 marks)
Phase 3 — Query
vi. ○ Open qryPassStudents. Set a criterion on Score to show only students with Score ≥ 50. Run the query. (5 marks)
vii. ○ Open qryTopStudents. Filter where Score ≥ 70 and sort by Score Descending. Run the query. (4 marks)
Phase 4 — Form
viii. ○ Open frmStudents. Add input fields for StudentID, FullName, Score and Grade. Add a Save Record button. (4 marks)
