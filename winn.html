15 — Architecture

3-Layer Architecture

Every agent works in exactly one layer. Layers do not share state — communicate via an event bus only. Layer 1 loads in under 2s on 3G. Layers 2 and 3 are lazy-loaded.

01
SURFACE
Win cards, stat blocks, step forms, newsreel, offline states, network badges. Everything works without JavaScript. Ships first. Always visible.
02
SECRET
3D brochure portal. Pocket dimension iframes. Hidden links to designer art. Easter eggs. Lazy-loaded on interaction. Zero performance impact on Layer 1.
03
PHYSICAL
GPS run tracker. WhatsApp deeplinks. Reward distribution. Social iframes. Require explicit user permission. Opt-in only.
16 — Architecture

Dormant Feature Index

These are built, commented out, and ready to switch on. One line uncomment per feature. Every agent must include these as properly-labeled comment blocks in their files.

// dormant_features — comment registry
/* MONETARY_REWARDS */
Multi-currency wallet — ₦/USDT/eNGN/SOL/[SECRET]. Connect, claim, withdraw. Activate when backend is live.
/* GPS_HEATMAP */
War-map win density. Mapbox GL. Amber→red heat. Real-time. Activate at volume.
/* POPUP_MODALS */
Win verify modal, reward claim modal, leaderboard modal, sprint announcement. Built, commented.
/* NEWSREEL_STACK */
3-lane live ticker: national / local / diaspora. Activate when feed volume justifies it.
/* VOTING_SYSTEM */
Weekly Win of the Week vote. Community co-sign. Anti-spam. Token reward for top voter.
/* SOCIAL_IFRAMES */
WhatsApp group counter, IG feed, X/Twitter feed. Activate per-channel as handles confirm.
/* AIRDROP_CLAIM */
Full eligibility check + wallet connect + claim flow. Lives behind 3D portal. Activate on token deploy.
/* SECRET_TOKEN_REVEAL */
Locked chest animation → token name + symbol + logo inject. Activate when identity is ready.
