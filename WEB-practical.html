Create a School Website Page
Using HTML and CSS, create a web page for JSS Model Secondary School. Complete all tasks in the editor. The preview updates in real time.
Structure
i.○Add the correct HTML boilerplate: <!DOCTYPE html>, <html>, <head>, <body> tags. (3 marks)
ii.○Set a <title> inside <head> — e.g. "JSS Model Secondary School". (2 marks)

Content
