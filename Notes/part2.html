10 · Coordinate Geometry — spatial / feeder node
x²+y²+2gx+2fy+c=0
Circle — extraction of centre & radius
Centre = (−g, −f). Radius = √(g²+f²−c). If coefficients aren't 1, divide the entire equation first — before anything else.
→ loci → tangent/normal problems centre is (−g, −f) not (+g, +f) if coefficient of x² ≠ 1, divide through FIRST
tricky question
Find the centre and radius of 2x² + 2y² − 8x + 12y − 6 = 0.
①Divide by 2: x² + y² − 4x + 6y − 3 = 0
②2g = −4 → g = −2. 2f = 6 → f = 3. c = −3
③Centre = (2, −3), r = √(4+9+3) = √16 = 4
Centre (2, −3), radius 4. Skipping the divide gives wrong centre and wrong radius.
tangent ⊥ radius at contact
Tangent to circle at a point
Step 1: find gradient of radius (centre to P). Step 2: tangent gradient = −1 ÷ radius gradient. Step 3: y − y₁ = m(x − x₁) using point P.
perpendicularity is the whole method — without it, there's no shortcut
tricky question
Find the tangent to x²+y²−4x+2y−20=0 at point (6, 2).
①Centre: (2, −1). Gradient of radius to (6,2): (2+1)/(6−2) = 3/4
②Tangent gradient = −4/3
③y−2 = −4/3(x−6) → 4x + 3y = 30
Tangent: 4x + 3y = 30.
let P=(x,y), write the condition
Loci — translate words to equations
Trigger: "locus of P such that...". Let P = (x,y). Express the condition algebraically. Simplify. Common results: straight line (equidistant), circle (fixed distance), parabola.
"equidistant from two points" → perpendicular bisector (line) "fixed distance from a point" → circle → complex number loci on Argand diagram
tricky question
Find the locus of P(x,y) equidistant from A(1,3) and B(5,−1).
①|PA|² = |PB|²: (x−1)²+(y−3)² = (x−5)²+(y+1)²
②Expand and cancel x², y²: −2x−6y+10 = −10x+2y+26
③8x − 8y = 16 → x − y = 2
Locus: x − y = 2. Verify: midpoint (3,1) satisfies 3−1=2 ✓
11 · Differentiation — hard / intuitive once seen
d/dx[f(g(x))] = f'(g)·g'
Chain rule — the invisible layer
Trigger: function inside a function — (3x+1)⁵, sin(x²), e^(2x). Differentiate outer, keep inner intact, multiply by derivative of inner.
d/dx[(2x+1)⁵] = 5(2x+1)⁴ × 2, not just 5(2x+1)⁴ → implicit differentiation → integration by substitution (reverse chain rule)
tricky question
Differentiate y = (x²+3)⁴ and find the gradient at x=1.
①dy/dx = 4(x²+3)³ · 2x = 8x(x²+3)³
②At x=1: 8(1)(4)³ = 8 × 64 = 512
Gradient = 512. Forgetting ×2x gives exactly half the answer.
d/dx[uv] = u'v + uv'
Product rule — two functions multiplied
Trigger: two distinct functions multiplied — x²sinx, xeˣ. Name u and v deliberately, write u' and v' beside them first, then assemble.
quotient rule: (u'v − uv')/v² — same family, different arrangement
tricky question
Find dy/dx if y = x³·eˣ. Where else (besides x=0) does dy/dx = 0?
①dy/dx = 3x²eˣ + x³eˣ = x²eˣ(3+x)
②eˣ ≠ 0 ever. x²=0 → x=0. x+3=0 → x=−3
Other zero at x = −3. The shock: eˣ never equals zero — students try to solve eˣ=0 and get stuck.
d²y/dx² < 0 → max, > 0 → min
Stationary points — second derivative test
Trigger: "maximum", "minimum", "turning point". Set dy/dx = 0, solve for x. Substitute into d²y/dx² to classify. If d²y/dx² = 0, use sign change of dy/dx instead.
if d²y/dx² = 0, the test is inconclusive — check sign of dy/dx on either side → optimisation word problems
tricky question
Find and classify the stationary points of y = x³ − 3x.
①dy/dx = 3x²−3 = 0 → x² = 1 → x = ±1
②d²y/dx² = 6x. At x=1: 6 > 0 → minimum. At x=−1: −6 < 0 → maximum
③y(1) = 1−3 = −2. y(−1) = −1+3 = 2
Maximum at (−1, 2), minimum at (1, −2).
12 · Integration — hardest / partial fractions feed here
∫f'(x)/f(x)dx = ln|f(x)|+C
Log integration — recognise numerator as derivative of denominator
Check: differentiate denominator — if you get numerator (up to a constant), apply this rule. Adjust by dividing through by that constant outside the integral.
← partial fractions (each term becomes this form) absolute value bars on ln are mandatory ∫1/(2x+1)dx = (1/2)ln|2x+1|+C, not ln|2x+1|+C
tricky question
Evaluate ∫(3x²+2)/(x³+2x+5) dx.
①Differentiate denominator: d/dx(x³+2x+5) = 3x²+2 ← exactly the numerator
②Directly: ln|x³+2x+5| + C
Answer: ln|x³+2x+5| + C. Students attempt substitution when recognition is the entire method.
∫u dv = uv − ∫v du
Integration by parts — LIATE for choosing u
Trigger: product of unlike functions — xsinx, x²eˣ, xlnx. LIATE order: Logarithm, Inverse trig, Algebraic, Trig, Exponential. First in list = u.
ln always becomes u — integrating ln directly requires parts anyway ∫x²eˣ needs parts applied TWICE
tricky question
Find ∫x·ln(x) dx.
①u = ln(x) → du = 1/x dx. dv = x dx → v = x²/2
②(x²/2)ln(x) − ∫(x²/2)(1/x)dx = (x²/2)ln(x) − ∫x/2 dx
③= (x²/2)ln(x) − x²/4 + C
Answer: (x²/2)ln(x) − x²/4 + C. Choosing u=x forces you to integrate ln(x) — which itself needs parts. LIATE prevents that loop.
area below x-axis → |∫|
Definite integrals & area — watch signs
Area below the x-axis gives a NEGATIVE result from the integral. For total area, split at x-intercepts and take absolute value of each section. Area between two curves: ∫(upper − lower)dx.
integrating straight across x-intercepts causes cancellation — wrong total area area between curves: always subtract, never add raw integrals
tricky question
Find the total area enclosed by y = x²−x and the x-axis between x=0 and x=2.
①Curve cuts x-axis: x(x−1)=0 → x=0 and x=1
②∫₀¹(x²−x)dx = −1/6 → area = 1/6
③∫₁²(x²−x)dx = 5/6
④Total = 1/6 + 5/6 = 1
Total area = 1 square unit. Integrating 0 to 2 directly gives 2/3 — the negative portion cancels part of the positive. Wrong answer, full working.
13 · Vectors — spatial / hard
a·b = |a||b|cosθ
Dot product — angle between vectors
Compute a·b = a₁b₁+a₂b₂+a₃b₃. Perpendicular: dot product = 0. Parallel: one is a scalar multiple of the other. Dot product is a scalar — never find its "unit vector".
negative dot product → obtuse angle. Not an error. a·b is SCALAR — no direction, no unit vector
tricky question
Find the angle between a = 2i+j−2k and b = i−2j+2k.
①a·b = 2−2−4 = −4. |a|=3, |b|=3
②cosθ = −4/9 → θ = cos⁻¹(−4/9) ≈ 116.4°
θ ≈ 116.4°. The negative result means obtuse angle — students panic and assume an error. There is none.
r = a + t(b−a)
Vector equation of a line
a = position vector of known point. (b−a) = direction. t = scalar parameter. To check if a point lies on a line: solve for t from one component, verify it satisfies all other components.
check ALL three components for the same t — checking only x is the common mistake direction vector can be any scalar multiple — all are valid
tricky question
Line through A(1,2,3) and B(3,−1,5). Does P(7,−7,9) lie on it?
①Direction: (2,−3,2). Line: (1,2,3)+t(2,−3,2)
②x: 1+2t=7 → t=3. y: 2−9=−7 ✓. z: 3+6=9 ✓
P lies on the line — t=3 consistent across all components.
14 · Probability — conditional & distributions
P(A|B) = P(A∩B)/P(B)
Conditional probability — the sample space shrinks
Trigger: "given that", "if it is known that". Everything must be within B now. P(A|B) ≠ P(B|A) — direction of conditioning matters completely.
← set theory (intersection) P(A|B) ≠ P(B|A) independent: P(A|B) = P(A) — conditioning changes nothing
tricky question
40% play football, 30% basketball, 15% both. Given a student plays football, find P(also plays basketball).
①P(B|F) = P(F∩B)/P(F) = 0.15/0.40
②= 15/40 = 3/8
P(B|F) = 3/8. Trap: answering 0.15 — that's P(F∩B), not P(B|F).
P(X=r) = ⁿCᵣ pʳ(1−p)ⁿ⁻ʳ
Binomial distribution — use complement for "at least"
Conditions: fixed n, binary outcomes, constant p, independent trials. "At least one" = 1 − P(X=0). Never compute "at least k" by summing directly — use 1 minus the complement.
← binomial theorem (same ⁿCᵣ) "at least one" → 1 − P(X=0). Direct sum takes too long and risks error. mean = np, variance = np(1−p)
tricky question
P(head) = 0.3. Coin tossed 6 times. Find P(at least 2 heads).
①P(X≥2) = 1 − P(X=0) − P(X=1)
②P(X=0) = 0.7⁶ = 0.117649
③P(X=1) = 6×0.3×0.7⁵ = 0.302526
④P(X≥2) = 1 − 0.420175 = 0.5798
P(X≥2) ≈ 0.5798. Complement = 2 calculations. Direct sum = 5. Always use complement.
15 · Word Problem Triggers — the recognition layer
read → classify → execute
Three steps before writing anything
The most dangerous moment is writing before classifying. Identify: (1) what is unknown, (2) what law governs it, (3) what form the answer must take. Then pick up the pen.
"maximum/minimum" → differentiate, set dy/dx = 0 "rate of change" → differentiation, chain rule likely "total accumulated / total distance" → integrate "how long until..." → set up equation in t
tricky question
A field fenced on THREE sides using 120m of wire. Find maximum area.
①Three sides: 2x+y=120 → y=120−2x
②A = x(120−2x) = 120x−2x²
③dA/dx = 120−4x = 0 → x=30, y=60
④d²A/dx² = −4 < 0 → maximum. A = 1800m²
Maximum area = 1800 m². Trap: using all four sides (perimeter=120) instead of three.
keyword → topic
Word-to-topic trigger map
Train your eye to see the topic before the numbers. Every word problem hides a topic underneath.
"installments / interest / investment" → GP or AP "arrangements / selections / committee" → permutation & combination "at random / probability / chance" → probability (check if conditional) "velocity / acceleration / displacement" → calculus "prove / show / for all n" → induction or proof "intersects / meets / common point" → simultaneous equations "remainder / last digit / divisible by" → modular arithmetic "angle between / perpendicular / parallel" → vectors or coordinate geometry
