01 · Partial Fractions — the universal feeder
A/(x+a) + B/(x+b)
Distinct linear factors
Trigger: denominator splits cleanly into two different linear factors. Substitute roots of each factor to find constants instantly — no simultaneous needed.
→ integration → binomial expansion if degree of numerator ≥ denominator, do long division FIRST
tricky question
Express (3x + 1)/[(x−1)(x+2)] in partial fractions.
①Write: (3x+1)/[(x−1)(x+2)] = A/(x−1) + B/(x+2)
②Multiply both sides by (x−1)(x+2): 3x+1 = A(x+2) + B(x−1)
③Let x=1: 4 = 3A → A = 4/3
④Let x=−2: −5 = −3B → B = 5/3
Answer: (4/3)/(x−1) + (5/3)/(x+2) — the non-integer constants are intentional. Don't doubt them.
A/(x+a) + B/(x+a)²
Repeated linear factor
Trigger: (x+a)² in denominator. You MUST include both powers. One equation won't suffice — equate coefficients after expanding.
writing only B/(x+a)² loses all marks for that fraction → integration of repeated factors
tricky question
Express (5x − 3)/[(x+1)(x−1)²] in partial fractions.
①Write: A/(x+1) + B/(x−1) + C/(x−1)²
②Multiply through: 5x−3 = A(x−1)² + B(x+1)(x−1) + C(x+1)
③x=1: 2 = 2C → C = 1
④x=−1: −8 = 4A → A = −2
⑤Equate x² coefficients: 0 = A + B → B = 2
Answer: −2/(x+1) + 2/(x−1) + 1/(x−1)²
A/(x+a) + (Bx+C)/(x²+b)
Irreducible quadratic factor
Trigger: denominator has x²+b that won't factorise over reals. Numerator of that fraction must be linear (Bx+C), not just a constant.
→ integration by completing the square Bx+C not just B — most common mark loss
tricky question
Express (x²+2)/[(x−1)(x²+1)] in partial fractions.
①Write: A/(x−1) + (Bx+C)/(x²+1)
②x²+2 = A(x²+1) + (Bx+C)(x−1)
③x=1: 3 = 2A → A = 3/2
④Equate x²: 1 = A+B → B = −1/2. Equate constants: 2 = A−C → C = −1/2
Answer: (3/2)/(x−1) + (−x/2 − 1/2)/(x²+1)
02 · Binomial Theorem — the great connector
(1+x)ⁿ = Σ ⁿCᵣ xʳ
General expansion — any n
Trigger: "expand", "coefficient of xʳ", "approximation". For non-integer or negative n, valid only when |x| < 1. Always rewrite (a+bx)ⁿ as aⁿ(1 + bx/a)ⁿ first.
→ probability distributions → approximation of surds/roots term r+1, not term r — "4th term" means r=3
tricky question
Find the coefficient of x² in the expansion of (2 − 3x)⁵.
①General term: ⁵Cᵣ · (2)⁵⁻ʳ · (−3x)ʳ
②For x²: r = 2 → ⁵C₂ · 2³ · (−3)² · x²
③= 10 · 8 · 9 · x² = 720x²
Coefficient = 720 — (−3)² = 9, not −9. Many students write −720.
(1+x)⁻¹ = 1−x+x²−x³+...
Negative & fractional index expansion
Trigger: "expand in ascending powers", "approximate" a root or reciprocal. Valid ONLY for |x| < 1. Coefficients follow n(n−1)(n−2).../r! pattern — they are NOT ⁿCᵣ when n is not a positive integer.
state the range of validity |x| < 1 — examiners always award a mark for this rewrite (1+2x)^(1/2): coefficients involve 1/2 × (−1/2) × (−3/2)... — work slowly
tricky question
Expand (1−2x)^(−2) up to and including the term in x³.
①n=−2. Coefficients: 1, (−2), (−2)(−3)/2!, (−2)(−3)(−4)/3!
②= 1, −2, 3, −4 (alternating, increasing magnitude)
③Replace x with (−2x): (1−2x)^(−2) = 1 + (−2)(−2x) + 3(−2x)² + (−4)(−2x)³ + ...
④= 1 + 4x + 12x² + 32x³ + ...
Answer: 1 + 4x + 12x² + 32x³, valid for |x| < 1/2. The shock: the coefficients grow, not shrink — students expect them to decrease.
03 · Sequences & Series — AP/GP as dependency nodes
Sₙ = n/2[2a + (n−1)d]
AP sum — and its commercial link
Trigger: "arithmetic progression", "common difference", "sum of first n terms". Simple interest produces an AP — each year adds a fixed amount. d = common difference = second term minus first.
→ commercial math (simple interest = AP) alternate form Sₙ = n/2(first + last) — use when last term is given
tricky question
The 3rd term of an AP is 8 and the 7th term is 20. Find the sum of the first 15 terms.
①a + 2d = 8 and a + 6d = 20. Subtract: 4d = 12 → d = 3
②a = 8 − 2(3) = 2
③S₁₅ = 15/2 [2(2) + 14(3)] = 15/2 [4+42] = 15/2 × 46 = 345
S₁₅ = 345. Trap: using n=15 in the term formula instead of the sum formula — two completely different things.
S∞ = a/(1−r), |r|<1
GP sum to infinity + recurring decimals
Trigger: "converges", "sum to infinity", OR any recurring decimal. A recurring decimal IS a geometric series. The recurring block is the first term; the ratio is a power of 1/10. Compound interest produces a GP.
→ commercial math (compound interest = GP) if |r| ≥ 1, series diverges — no sum to infinity
tricky question
Express 0.727272... as a fraction.
①Recurring block "72" → a = 72/100, r = 1/100
②S∞ = (72/100) / (1 − 1/100) = (72/100)/(99/100) = 72/99
③HCF(72,99) = 9 → 8/11
Answer: 8/11 — most students never recognise this as a GP problem.
04 · Mathematical Induction — shocking / proof
n=1 → assume k → prove k+1
The three-step lock — never skip step 2
Step 2 must be written explicitly: "Assume true for n=k, i.e. [write the formula with k]." Examiners mark for this line. Step 3 must start FROM that assumption.
conclusion must say "true for all positive integers n by mathematical induction" ← series summation
tricky question
Prove that 5ⁿ − 1 is divisible by 4 for all positive integers n.
①n=1: 5¹−1 = 4. Divisible by 4. ✓
②Assume true for n=k: 5ᵏ−1 = 4m for some integer m, so 5ᵏ = 4m+1
③For n=k+1: 5ᵏ⁺¹−1 = 5·5ᵏ−1 = 5(4m+1)−1 = 20m+4 = 4(5m+1)
Key: substitute 5ᵏ = 4m+1 from step ②. Students who restart from scratch get zero for step 3.
05 · Number Bases — shocking / mode-switch
carry at n, not at 10
Arithmetic in non-decimal bases
Trigger: subscript notation 132₅, "evaluate in base n". Every column resets at the base. In base 5: 3+4 = 12₅ (write 2, carry 1).
never convert to base 10, compute, convert back — work IN the base borrow = gain one full base value
tricky question
Evaluate 213₄ + 132₄, giving your answer in base 4.
①Units: 3+2=5. 5÷4=1 rem 1. Write 1, carry 1
②Fours: 1+3+1(carry)=5. 5÷4=1 rem 1. Write 1, carry 1
③Sixteens: 2+1+1(carry)=4. 4÷4=1 rem 0. Write 0, carry 1
④Final carry: write 1
Answer: 1011₄. Check: 213₄ = 2(16)+1(4)+3 = 39. 132₄ = 1(16)+3(4)+2 = 30. 39+30=69=1(64)+0(16)+1(4)+1 = 1011₄ ✓
06 · Logic & Truth Tables — shocking / mode-switch
p → q ≡ ¬p ∨ q
Implication — the one false row
False ONLY when p is TRUE and q is FALSE. Every other combination is TRUE. Contrapositive ¬q→¬p is always logically equivalent. Converse q→p is NOT equivalent.
T→F = F. This is the only false case. Students flip this constantly. converse ≠ contrapositive
tricky question
Construct a truth table for (p → q) ∧ (q → p) and state what it represents.
①p→q row by row (TT,TF,FT,FF): T, F, T, T
②q→p: T, T, F, T
③AND of both: T, F, F, T — true only when p and q match
This is p ↔ q (biconditional) — "p if and only if q". The AND of two implications IS a biconditional.
¬(p∧q) = ¬p∨¬q
De Morgan's laws
Trigger: negation of a compound statement. Distribute the NOT and flip AND↔OR. Applies identically to set theory: (A∩B)' = A'∪B'.
→ set theory (same structure) the AND/OR flip is mandatory — the most common error is distributing NOT without flipping
tricky question
Negate: "It is raining AND the road is wet."
①Let p = "It is raining", q = "the road is wet". Statement: p ∧ q
②Negation: ¬(p∧q) = ¬p ∨ ¬q
Answer: "It is NOT raining OR the road is NOT wet." — NOT both. Students write "It is not raining AND the road is not wet" — that's ¬p ∧ ¬q, which is stronger and wrong.
07 · Modular Arithmetic — shocking
a ≡ b (mod n)
Congruence — same remainder, think clocks
a and b leave the same remainder when divided by n. You can add, subtract, multiply congruences directly. For large powers: find the cycle of remainders, then use the position in the cycle.
congruence ≠ equality division requires modular inverse — never just divide both sides
tricky question
Find the remainder when 7²⁰ is divided by 5.
①7 ≡ 2 (mod 5), so 7²⁰ ≡ 2²⁰ (mod 5)
②Powers of 2 mod 5: 2,4,3,1,2,4,3,1... period = 4
③20 mod 4 = 0 → position 4 in cycle → 2⁴ ≡ 1 (mod 5)
Remainder = 1. The cyclic pattern is the entire technique — computing 7²⁰ directly is impossible.
08 · Complex Numbers — shocking / mode-switch
i² = −1, cycle: i,−1,−i,1
Powers of i — cycle of 4
For iⁿ: find n mod 4, match to cycle. Division: multiply top and bottom by conjugate of denominator. Argand diagram: real axis horizontal, imaginary vertical.
← quadratics with negative discriminant → Argand diagram loci conjugate of a+bi is a−bi — flips imaginary sign only
tricky question
Express (2+3i)/(1−2i) in the form a+bi.
①Multiply by (1+2i)/(1+2i)
②Numerator: (2+3i)(1+2i) = 2+4i+3i+6i² = 2+7i−6 = −4+7i
③Denominator: (1)²+(2)² = 5
Answer: −4/5 + (7/5)i. Trap: forgetting i²=−1 in the numerator flips the sign of 6.
|z|=√(a²+b²), arg=arctan(b/a)
Modulus and argument — watch the quadrant
Modulus = distance from origin. Argument = angle from positive real axis, measured anticlockwise. arctan gives Q1/Q4 angle only — adjust manually for Q2 and Q3.
Q2: arg = π − arctan(|b/a|). Q3: arg = −π + arctan(|b/a|) argument range is usually (−π, π] — check what the question asks
tricky question
Find the modulus and argument of z = −3 + 3i.
①|z| = √(9+9) = √18 = 3√2
②arctan(3/3) = arctan(1) = 45° = π/4
③z is in Q2 (negative real, positive imaginary) → arg = π − π/4 = 3π/4
|z| = 3√2, arg = 3π/4. Trap: calculators give arctan(−1) = −π/4 if you enter −3/3 directly — wrong quadrant entirely.
09 · Matrices — hard / procedural
A⁻¹ = (1/det)[d −b; −c a]
Inverse to solve simultaneous equations
Write AX = B, solve as X = A⁻¹B. Order matters — matrix multiplication is not commutative. If det = 0, no unique solution exists (singular matrix).
← simultaneous equations → transformations (|det| = area scale factor) swap a↔d, negate b and c — students negate all four entries X = A⁻¹B ≠ BA⁻¹
tricky question
Solve using matrices: 3x + y = 7, x − 2y = 0.
①A = [3 1; 1 −2], B = [7; 0]
②det(A) = (3)(−2)−(1)(1) = −7
③A⁻¹ = (−1/7)[−2 −1; −1 3]
④X = A⁻¹B = (−1/7)[−14; −7] = [2; 1]
x = 2, y = 1. Trap: negative determinant — dividing by −7 not 7 changes all signs.
