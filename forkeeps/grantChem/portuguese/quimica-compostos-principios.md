# QUÍMICA DOS COMPOSTOS

A química dos compostos começa onde o Átomo parou: `composto = molécula + molécula`. Mas esta única equação esconde cinco personalidades completamente diferentes, e quase todos os programas ensinam-nas como cinco temas sem relação em vez de cinco ramos da mesma bifurcação.

```
PORTA → Ácido / Base / Sal / Neutro-Anfotérico / Complexo (liga/polímero/composto de coordenação)
```

Nada neste capítulo escapa a esta primeira bifurcação. Tudo o que se segue é o que acontece depois de já se ter respondido a essa pergunta.

---

## 0. ÓXIDOS — a ponte que ninguém trata como ponte

Antes de o ácido e a base poderem reagir um com o outro, chegam normalmente sob a forma de óxidos — um elemento mais oxigénio, à porta, a decidir qual o ramo a que se vai juntar.

### A Porta

```
PORTA → Óxido ácido / Óxido básico / Óxido anfotérico / Óxido neutro
```

| Tipo de óxido | Formado por | Reage com | Exemplo |
|---|---|---|---|
| Ácido | Não-metal + oxigénio | Base (forma sal + água) | CO2, SO2 |
| Básico | Metal + oxigénio | Ácido (forma sal + água) | Na2O, CaO |
| Anfotérico | Certos metais (caráter metálico fronteiriço) | Ácido e base | Al2O3, ZnO |
| Neutro | Nem plenamente metálico nem reativo | Nenhum | CO, H2O, NO |

Frase-síntese: **um óxido é a frase por terminar de um elemento — diz-nos a que ramo (ácido ou base) o elemento está prestes a juntar-se, antes mesmo da reação que o prova.** O óxido de um não-metal a dissolver-se em água já é um ácido disfarçado (CO2 + H2O → H2CO3); o óxido de um metal a dissolver-se em água já é uma base disfarçada (CaO + H2O → Ca(OH)2). O passo do óxido não é um tema separado — é a porta ácido/base, um passo mais cedo do que o habitual.

## I. ÁCIDO — o dador de protões

### Classificação por múltiplas lentes

| Lente | Pergunta | Divisão |
|---|---|---|
| Força | Quão completamente se dissocia em água? | Forte (totalmente) / Fraco (parcialmente, equilíbrio) |
| Basicidade | Quantos H+ substituíveis carrega? | Monoprótico (1) / Diprótico (2) / Triprótico (3) |
| Origem | De onde vem o ácido? | Mineral/inorgânico (HCl, H2SO4) / Orgânico (etanoico, cítrico) |
| Conteúdo de oxigénio | Contém oxigénio? | Oxiácido (H2SO4, HNO3) / Hidrácido (HCl, HBr) |

Quatro respostas verdadeiras para o mesmo ácido, a mesma lógica das três lentes do sal no Livro 3.

### Ácidos comuns, nomeados e situados

| Ácido | Força | Basicidade | Nome comum / onde aparece |
|---|---|---|---|
| Clorídrico (HCl) | Forte | Mono | Ácido do estômago |
| Sulfúrico (H2SO4) | Forte | Di | Baterias de automóvel |
| Nítrico (HNO3) | Forte | Mono | Produção de fertilizantes |
| Etanoico (acético, CH3COOH) | Fraco | Mono | Vinagre — um ácido fraco com nome de cozinha |
| Ácido cítrico | Fraco | Tri | O azedume dos citrinos |
| Carbónico (H2CO3) | Fraco | Di | Bebidas gaseificadas — CO2 dissolvido em água, a formar ácido em tempo real |

### Operações — o que um ácido faz, como procedimentos

```
ácido + metal          → sal + hidrogénio gasoso        (teste: o gás estala com uma chama)
ácido + base           → sal + água                      (neutralização)
ácido + carbonato      → sal + água + CO2                 (teste: o gás turva a água de cal)
ácido + óxido metálico → sal + água
```

O mesmo ácido, quatro parceiros diferentes, uma identidade confirmada de cada vez por qualquer gás ou produto que apareça. O teste nunca é separado da reação — o teste *é* a reação, lida pela sua assinatura.

## II. BASE — o aceitador de protões

### A distinção que a maioria confunde

**Base** é qualquer aceitador de protões. **Álcali** é uma base que é *solúvel em água*. Todo o álcali é uma base; nem toda a base é um álcali — bases insolúveis, como o hidróxido de cobre(II), nunca tocam na água, mas ainda assim neutralizam um ácido ao contacto.

| Termo | Relação |
|---|---|
| Base | Aceita um protão — a categoria geral |
| Álcali | Uma base que se dissolve — o subconjunto que se pode titular diretamente |

### Bases comuns

| Base | É álcali? | Força |
|---|---|---|
| NaOH | Sim | Forte |
| KOH | Sim | Forte |
| Ca(OH)2 | Ligeiramente solúvel — fronteiriço | Forte (no que se dissolve) |
| NH3 (solução de amoníaco) | Sim | Fraca |
| Cu(OH)2, Fe(OH)3 | Não | Insolúvel, ainda assim básico |

### Operações

```
base + ácido                        → sal + água
álcali + sal de amónio, aquecido    → sal + água + gás NH3   (teste: torna azul o papel de tornesol vermelho húmido)
```

## III. SAL — o resto

O sal já recebeu a sua relação no Livro 3: **o que sobra quando o ácido dá, a base aceita e a água se vai embora com a diferença.** Esta secção é onde esse resto é organizado.

### Classificação por múltiplas lentes (alargada)

| Lente | Divisão | Lógica |
|---|---|---|
| Composição | Normal / Ácido / Básico / Duplo / Complexo | Quão completamente o H+ do ácido (ou o OH– da base) foi substituído |
| Origem | Mineral / Orgânico | A mesma lente dos ácidos, herdada |
| Solubilidade | Solúvel / Insolúvel | Decide o método de preparação — ver abaixo |
| Hidratação | Anidro / Hidratado (carrega água de cristalização) | Se as moléculas de água estão fixas na estrutura cristalina |

### Tipos de sal, por composição

| Tipo | O que aconteceu | Exemplo |
|---|---|---|
| Normal | Todo o H+ (ou OH–) substituível foi completamente trocado | NaCl, a partir de HCl + NaOH |
| Ácido | Nem todo o H+ foi substituído — mantém-se algum caráter ácido | NaHSO4, NaHCO3 |
| Básico | Nem todo o OH– foi substituído — mantém-se algum caráter básico | Carbonato básico de chumbo |
| Duplo | Dois sais diferentes a cristalizar juntos numa proporção fixa | Alúmen de potássio, KAl(SO4)2·12H2O |
| Complexo | Um ião metálico rodeado de ligandos, a unidade toda carregada | [Cu(NH3)4]SO4 |

Frase-síntese: **um sal ácido é um ácido que nunca terminou a neutralização — a mesma lógica de assunto por terminar do radical que nunca terminou de deixar o seu ião.** O padrão do Livro 3 (radical = ião por terminar) reaparece aqui como (sal ácido = ácido por terminar).

### O método de preparação depende inteiramente da solubilidade — é aqui que a maioria decora receitas separadas em vez de uma só árvore de decisão

```
PORTA → O sal alvo é solúvel?
   SIM → titulação (ácido + álcali, indicador) ou ácido em excesso + base/carbonato/metal insolúvel, depois filtrar o excesso e cristalizar
   NÃO → precipitação (dupla decomposição) — misturar dois sais solúveis, o sal insolúvel precipita, filtrar
```

| Método | Usado quando | Exemplo |
|---|---|---|
| Titulação | Sal solúvel a partir de ácido + base solúvel | NaCl a partir de HCl + NaOH |
| Sólido em excesso + ácido, filtrar, cristalizar | Sal solúvel a partir de ácido + base/carbonato/metal insolúvel | CuSO4 a partir de H2SO4 + excesso de CuO |
| Precipitação (dupla decomposição) | Sal insolúvel | PbCl2 a partir de Pb(NO3)2 + NaCl |
| Combinação direta | Sal a partir dos elementos diretamente | FeCl3 a partir de Fe + Cl2 |

Toda a tabela de métodos colapsa numa só pergunta: o sal que se quer fica dissolvido ou precipita? Tudo o resto é apenas qual reagente é conveniente ter em excesso.

### Água de cristalização

```
sal hidratado --(calor)--> sal anidro + água (libertada como vapor)
```

Os cristais azuis de sulfato de cobre(II) (CuSO4·5H2O) a tornarem-se pó branco ao aquecer é a prova visível: a cor azul pertence à estrutura ligada à água, não ao sal sozinho. Juntar água de novo ao pó branco e o azul regressa — esta reversibilidade é o teste padrão para a água.

### A Escala do Sal — o eixo que o pH não mostra

O pH mede o que fica dissolvido em solução — diz o caráter do meio. Nunca diz o que precipitou desse meio, ou o que se formou e se afastou como substância separada. É um eixo completamente diferente, e corre verticalmente, exatamente através do ponto neutro do pH em 7, em vez de ao longo dele.

```
                     ÉSTERES
                (condensação de ácido orgânico + álcool,
                 ou lípido + glicerol — água libertada,
                 a resposta orgânica ao "sal")
                        ↑
                      SABÃO
              (sal orgânico — saponificação,
               éster + base forte → sal + glicerol)
                        ↑
                  SAL COMPLEXO
              (ião metálico + ligandos, unidade carregada)
                        ↑
                    SAL DUPLO
            (dois sais a cristalizar numa proporção fixa)
                        ↑
                    SAL BÁSICO
              (algum OH⁻ por substituir)
━━━━━━━━━━━━ SAL NORMAL ━━━━━━━━━━━━  ← situa-se no eixo neutro do pH, o caso canónico
              (ácido totalmente + base totalmente → sal + água)
                        ↓
                    SAL ÁCIDO
              (algum H⁺ por substituir)
                        ↓
               PRECIPITADO MINERAL
        (sal inorgânico insolúvel, formado por
         dupla decomposição, não por neutralização —
         BaSO4, AgCl, CaCO3)
                      FUNDO
```

Ler de baixo para cima: no fundo está o sal mais mineral, menos "escolhido" — um precipitado que simplesmente saiu do encontro de duas soluções, sem ácido a dar um protão, sem base a aceitá-lo, apenas a insolubilidade a fazer o trabalho (as regras de solubilidade da Secção III, não neutralização nenhuma). No topo está o éster — não é tecnicamente um sal pela definição estrita de ácido+base, mas construído pela mesma lógica exata de condensação *dador-encontra-aceitador-liberta-água*, apenas com um álcool a substituir a base e um ácido gordo a substituir o ácido mineral.

O sal normal, no meio, é o único ponto desta escala que se situa diretamente no eixo neutro do pH — tudo acima e abaixo dele continua a ser "o resto de uma reação de dar e receber," só que cada vez mais orgânico a subir, e cada vez mais mineral-bruto a descer.

Frase-síntese: **o pH diz o que ainda está em solução. A Escala do Sal diz o que saiu dela — e quanto mais essa saída se parecer com biologia (lípidos, ésteres, fragrância) em vez de geologia (precipitados, minerais), mais alto se sobe nesta escala.** O sabão situa-se exatamente onde seria de esperar, dada a equação do sabão da Secção VIII — um sal orgânico, mais perto do extremo do éster do que do extremo mineral, porque a saponificação está a um passo de condensação de um verdadeiro éster.

## IV. NEUTRO E ANFOTÉRICO — a mesma substância, cara diferente consoante o que enfrenta

| Termo | Comportamento |
|---|---|
| Neutro | Não dá nem aceita um protão — genuinamente inerte a ácido e base (água pura, solução de sal de cozinha) |
| Anfotérico | Dá a uma base, aceita de um ácido — a mesma molécula a trocar de papel consoante com quem reage |

```
óxido/hidróxido anfotérico + ácido  → comporta-se como base → sal + água
óxido/hidróxido anfotérico + álcali → comporta-se como ácido → sal + água
```

Al2O3 e Al(OH)3 são os exemplos padrão — o mesmo composto de alumínio dissolve-se tanto em ácido clorídrico como em hidróxido de sódio, o que é exatamente a razão pela qual objetos de alumínio não devem ser limpos nem com ácidos fortes nem com álcalis fortes: não tem lado nenhum onde esteja seguro.

---

## V. HIDRÓLISE DOS SAIS — o pH de um sal não é automaticamente 7

O pH de um sal dissolvido depende da força do ácido e da base que o formaram — e esta é a operação que a maioria salta diretamente para decorar "os sais são neutros," o que só é verdade um quarto das vezes.

| Força do ácido | Força da base | pH da solução salina resultante |
|---|---|---|
| Forte | Forte | Neutro (NaCl — genuinamente pH 7) |
| Forte | Fraca | Ácido (NH4Cl — o conjugado da base fraca não consegue manter posição) |
| Fraco | Forte | Básico (CH3COONa — o conjugado do ácido fraco vence o cabo de guerra) |
| Fraco | Fraca | Depende de qual é relativamente mais forte — raramente testado a este nível |

Frase-síntese: **o pH de um sal é apenas um placar do resto — quem quer que tenha sido o progenitor (ácido ou base) mais fraco perde primeiro o controlo, e o caráter do progenitor sobrevivente mais forte aparece na solução.**

## VI. ANÁLISE QUALITATIVA — identificar sais por operação, não por adivinhação

### Testes de catiões (juntar NaOH, depois em excesso se necessário)

| Catião | Com NaOH (algumas gotas) | Com NaOH em excesso |
|---|---|---|
| Fe2+ | Precipitado verde | Insolúvel, mantém-se verde |
| Fe3+ | Precipitado castanho-avermelhado | Insolúvel, mantém-se castanho-avermelhado |
| Cu2+ | Precipitado azul | Insolúvel, mantém-se azul |
| Zn2+ | Precipitado branco | Dissolve-se — solução incolor (anfotérico) |
| Al3+ | Precipitado branco | Dissolve-se — solução incolor (anfotérico) |
| NH4+ | Sem precipitado | Aquecer suavemente → gás amoníaco (torna azul o papel de tornesol vermelho húmido) |

### Testes de aniões

| Anião | Teste | Resultado |
|---|---|---|
| Carbonato (CO3^2-) | Juntar ácido diluído | Efervescência, o gás turva a água de cal |
| Sulfato (SO4^2-) | Juntar BaCl2 + HCl diluído | Precipitado branco (BaSO4), insolúvel em ácido |
| Cloreto (Cl–) | Juntar AgNO3 + HNO3 diluído | Precipitado branco (AgCl) |
| Brometo (Br–) | Juntar AgNO3 + HNO3 diluído | Precipitado creme (AgBr) |
| Iodeto (I–) | Juntar AgNO3 + HNO3 diluído | Precipitado amarelo (AgI) |
| Nitrato (NO3–) | Teste do anel castanho (Fe2+ + H2SO4 concentrado, em camadas) | Anel castanho na interface |

O padrão que vale a pena nomear: **a maioria dos testes qualitativos são apenas as regras de solubilidade da Secção III, ao contrário — não se está a descobrir química nova, está-se a usar uma insolubilidade conhecida para pescar um ião desconhecido.**

## VII. COMPLEXO / LIGA / POLÍMERO — o quinto ramo

Este ramo não se forma por troca ácido-base — forma-se por moléculas ou iões que simplesmente concordam em associar-se, e é por isso que está à parte de ácido/base/sal/neutro como categoria própria.

### Compostos de coordenação

```
ião complexo = ião metálico central + ligandos (dadores que partilham um par isolado, não uma ligação completa)
```

| Termo | Relação |
|---|---|
| Ligando | Uma molécula ou ião que empresta eletrões sem ligar por completo da forma que um parceiro covalente faria — um convidado, não um cônjuge |
| Número de coordenação | Quantos ligandos rodeiam o ião central — normalmente 4 ou 6 |
| Sal complexo | Um ião complexo emparelhado com um contraião para equilibrar a carga — [Cu(NH3)4]SO4 |

### Ligas — um composto que não é bem um composto

Uma liga é uma mistura, não um composto no sentido estrito da ligação — átomos metálicos de tamanhos diferentes compactados numa só rede, sem fórmula fixa, sem estequiometria.

| Liga | Composição | Porque é melhor que o metal puro |
|---|---|---|
| Aço | Ferro + carbono | Os átomos de carbono encravam a rede de ferro, bloqueando as camadas que deixam o ferro puro dobrar-se facilmente — mais duro, menos maleável |
| Latão | Cobre + zinco | A mesma lógica de encravamento de rede — mais duro do que qualquer um dos metais puros |
| Duralumínio | Alumínio + cobre | A leveza do alumínio, resistência próxima do aço |

Frase-síntese: **uma liga fica mais forte pela mesma razão que um hidrocarboneto ramificado ferve a temperatura mais baixa — partes de tamanhos diferentes perturbam um padrão que uma compactação pura e uniforme permitiria.** A lógica do encravamento de rede e a lógica da ramificação do Livro 3 são a mesma ideia física, vestida com dois nomes de disciplina diferentes.

### Polímeros, revisitados

Já cobertos em Orgânica (Secção V) como adição/condensação. Vale a pena recapitular aqui só pela posição: um polímero pertence a este ramo porque, tal como um complexo, é moléculas a associarem-se em massa em vez de ácido e base a trocarem protões.

## VIII. TÉCNICAS DE SEPARAÇÃO, ANÁLISE E SEGURANÇA PRÁTICA

Toda a técnica de separação explora exatamente uma diferença física entre as coisas a separar. Aprender a propriedade, e a técnica deixa de ser uma lista decorada e passa a ser uma consulta.

### A Porta

```
PORTA → Que propriedade difere entre os componentes?
   Tamanho de partícula (dissolvido vs não dissolvido) → Filtração
   Ponto de ebulição, mesma fase da mistura              → Destilação (simples: 1 líquido + sólido dissolvido; fracionada: 2+ líquidos, pontos de ebulição próximos)
   Solubilidade a temperaturas diferentes                 → Cristalização
   Densidade, líquidos imiscíveis                         → Funil de decantação
   Solubilidade num solvente em movimento (mobilidade)     → Cromatografia
   Sólido → gás diretamente, sem fase líquida              → Sublimação
```

| Técnica | Propriedade explorada | Exemplo |
|---|---|---|
| Filtração | Tamanho de partícula | Areia da água |
| Destilação simples | Ponto de ebulição, sólido vs líquido | Sal da água |
| Destilação fracionada | Ponto de ebulição, líquido vs líquido | Etanol da água; petróleo bruto (Orgânica, Secção VI) |
| Cristalização | A solubilidade desce com a temperatura | Cristais de sulfato de cobre a partir de solução |
| Funil de decantação | Densidade, imiscibilidade | Óleo da água |
| Cromatografia | Mobilidade diferencial no solvente | Separar os componentes de um corante/tinta |
| Sublimação | Transição direta de sólido a gás | Iodo, cloreto de amónio numa mistura |

Frase-síntese: **não existe um "capítulo de separação" distinto de tudo o que veio antes — cada técnica é apenas a porta da solubilidade da Secção III ou a tendência do ponto de ebulição da Secção I, transformada em procedimento manual.**

---

### Regras de solubilidade — a tabela de que toda a decisão sobre sais acima dependia

Esta é a tabela que a porta de método de preparação da Secção III já pressupunha decorada. Enunciá-la aqui, corretamente, de uma vez por todas:

| Ião / família de sal | Solubilidade | Exceções (insolúveis) |
|---|---|---|
| Sais do Grupo 1 (Na+, K+) e de amónio (NH4+) | Sempre solúveis | Nenhuma |
| Nitratos (NO3–) | Sempre solúveis | Nenhuma |
| Cloretos, brometos, iodetos | Geralmente solúveis | AgCl, PbCl2, Hg2Cl2 (e os equivalentes de Br/I) — insolúveis |
| Sulfatos (SO4^2-) | Geralmente solúveis | BaSO4, PbSO4, CaSO4 (ligeiramente) — insolúveis |
| Carbonatos (CO3^2-) | Geralmente insolúveis | Carbonatos do Grupo 1 e carbonato de amónio — solúveis |
| Hidróxidos (OH–) | Geralmente insolúveis | Hidróxidos do Grupo 1 — solúveis; Ca(OH)2 — ligeiramente solúvel |

Ler esta tabela antes de escolher um método de preparação, não depois: **se tanto o sal alvo como o provável parceiro de dupla decomposição forem solúveis, a titulação é a rota; se o próprio sal estiver numa coluna de "exceções," a precipitação é a única rota que alguma vez vai funcionar.**

---

### Titulação — fazer corresponder o indicador ao par ácido-base, não adivinhar

Um indicador só funciona se mudar de cor exatamente ao pH onde o ponto de equivalência realmente se situa — usar o errado dá um ponto final falso, mesmo que a mudança de cor pareça convincente.

| Par ácido + base | pH do ponto de equivalência | Indicador correto |
|---|---|---|
| Ácido forte + Base forte | ~7 (neutro) | Fenolftaleína ou alaranjado de metilo |
| Ácido forte + Base fraca | Ácido (abaixo de 7) | Só alaranjado de metilo |
| Ácido fraco + Base forte | Básico (acima de 7) | Só fenolftaleína |
| Ácido fraco + Base fraca | Sem ponto final nítido | Evitar titular este par — nenhum indicador dá uma leitura limpa |

Esta tabela é a lógica da hidrólise da Secção V, aplicada em sentido inverso: ali, previa-se o pH de um sal a partir do ácido/base progenitor; aqui, escolhe-se o indicador que vai realmente apanhar a mudança de cor nesse mesmo pH previsto.

### O que é seguro reagir, e o que não é

```
REGRA 1: Juntar sempre o ácido à água, nunca a água ao ácido concentrado.
         (Diluir ácido concentrado liberta calor de repente — juntar água ao ácido deixa uma
         pequena quantidade de água ferver instantaneamente à superfície e cuspir ácido de volta.
         Juntar ácido a um volume maior de água espalha o calor em vez de o concentrar num só ponto.)

REGRA 2: Nunca misturar lixívia (hipoclorito) com ácido — liberta gás cloro tóxico.

REGRA 3: Nunca aquecer um recipiente selado. O gás produzido por qualquer reação precisa
         de uma saída, ou o recipiente torna-se a explosão.

REGRA 4: Testar a identidade do gás antes de o assumir — um estalido agudo (H2), uma vareta
         em brasa a reacender (O2) e a água de cal a turvar (CO2) são gases diferentes com
         necessidades de manuseamento diferentes, não são "algum gás apareceu" intermutáveis.
```

Frase-síntese: **a maioria das regras de segurança no laboratório não são cautela arbitrária — são as mesmas operações de reação das Secções I, II e III, apenas a decorrer a uma escala ou velocidade onde o subproduto (calor, gás, pressão) deixa de ser uma observação arrumada e passa a ser um perigo.**

```
   ↓                        ↓                     ↓                             ↓                          de rede em ligas

                                              análise qualitativa                                           para Orgânica, Secção V)
```

A química dos compostos não são cinco temas. É uma bifurcação, cinco direções, e cada reação "nova" dentro de cada direção é a mesma lógica ácido-base-sal, colocada mais uma vez com um parceiro diferente na sala.

Separação, solubilidade e segurança no laboratório não são um sexto tema colado no fim — são a mesma bifurcação, tornada física: filtração e cristalização são a porta da solubilidade com as mãos dentro dela, a escolha de indicador de titulação é a tabela de hidrólise lida ao contrário, e toda a regra de segurança é uma operação de reação a decorrer a uma escala onde o subproduto deixa de ser observação e passa a ser perigo.

## ANTES DE FECHAR ESTE CAPÍTULO

Não é um resumo. É um ponto de controlo. Se alguma destas cinco perguntas travar, é esse o bolso a reabrir — não o capítulo inteiro.

1. Um óxido metálico e um óxido não-metálico caem ambos em água. Sem nomear nenhum dos compostos, a que família pertence cada solução, e porque é que a identidade do elemento sozinha já decidiu isso?
2. Recebe-se um sólido branco e diz-se apenas que é um sal. Nomeie os dois resultados de laboratório que, sozinhos, diriam se se deve preparar mais dele por titulação ou por precipitação.
3. Um sal testa neutro em solução. Enuncie a única coisa que agora se sabe com certeza sobre o ácido e a base que o formaram — e a única coisa que ainda não se sabe.
4. A água de bromo descolora uma amostra de hidrocarboneto. Que ligação quebrou para causar isso, e o que diz a ausência dessa mesma ligação no benzeno sobre a razão pela qual os compostos aromáticos resistem exatamente a este teste?
5. Alguém entrega ácido sulfúrico concentrado e um copo de água e pergunta qual vai para dentro de qual primeiro. Responda numa frase, e numa segunda frase diga o que acontece se se fizer ao contrário.

Se a pergunta cinco foi aquela em que se hesitou: isso não é uma falha de memória. É a única pergunta neste capítulo em que errar não custa um ponto — custa a camada superior da pele da mão. Alguns bolsos deste livro é inconveniente esquecer. Esse não tem permissão para o ser.
