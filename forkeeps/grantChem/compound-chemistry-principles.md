# COMPOUND CHEMISTRY

Compound chemistry starts where Atom left off: `compound = molecule + molecule`. But that single equation hides five entirely different personalities, and almost every syllabus teaches them as five unrelated topics instead of five branches off one fork.

```
GATE → Acid / Base / Salt / Neutral-Amphoteric / Complex (alloy/polymer/coordination)
```

Nothing in this chapter escapes that first fork. Everything below is what happens after you've already answered it.

---

## 0. OXIDES — the bridge nobody names as a bridge

Before acid and base can react with each other, they usually arrive as oxides — an element plus oxygen, standing at the door deciding which branch to join.

### The Gate

```
GATE → Acidic oxide / Basic oxide / Amphoteric oxide / Neutral oxide
```

| Oxide type | Made by | Reacts with | Example |
|---|---|---|---|
| Acidic | Nonmetal + oxygen | Base (forms salt + water) | CO2, SO2 |
| Basic | Metal + oxygen | Acid (forms salt + water) | Na2O, CaO |
| Amphoteric | Certain metals (borderline metallic character) | Both acid and base | Al2O3, ZnO |
| Neutral | Neither fully metallic nor reactive character | Neither | CO, H2O, NO |

Relationship one-liner: **an oxide is an element's unfinished sentence — it tells you which branch (acid or base) the element is about to join, before the reaction that proves it.** A nonmetal's oxide dissolving in water is already an acid in disguise (CO2 + H2O → H2CO3); a metal's oxide dissolving in water is already a base in disguise (CaO + H2O → Ca(OH)2). The oxide step is not a separate topic — it's the acid/base gate, one step earlier than usual.

## I. ACID — the proton donor

### Multi-lens classification

| Lens | Question | Split |
|---|---|---|
| Strength | How completely does it dissociate in water? | Strong (fully) / Weak (partially, equilibrium) |
| Basicity | How many replaceable H+ does it carry? | Monoprotic (1) / Diprotic (2) / Triprotic (3) |
| Origin | Where does the acid come from? | Mineral/inorganic (HCl, H2SO4) / Organic (ethanoic, citric) |
| Oxygen content | Does it contain oxygen? | Oxyacid (H2SO4, HNO3) / Hydracid (HCl, HBr) |

Four true answers for the same acid, same logic as salt's three lenses in Book 3.

### Common acids, named and placed

| Acid | Strength | Basicity | Common name / where it shows up |
|---|---|---|---|
| Hydrochloric (HCl) | Strong | Mono | Stomach acid |
| Sulfuric (H2SO4) | Strong | Di | Car batteries |
| Nitric (HNO3) | Strong | Mono | Fertilizer production |
| Ethanoic (acetic, CH3COOH) | Weak | Mono | Vinegar — a weak acid wearing a kitchen name |
| Citric acid | Weak | Tri | Citrus fruit sourness |
| Carbonic (H2CO3) | Weak | Di | Fizzy drinks — CO2 dissolved in water, forming acid in real time |

### Operations — what an acid does, stated as procedures

```
acid + metal          → salt + hydrogen gas         (test: gas pops with a lit splint)
acid + base           → salt + water                 (neutralization)
acid + carbonate      → salt + water + CO2            (test: gas turns limewater cloudy)
acid + metal oxide    → salt + water
```

Same acid, four different partners, one identity confirmed each time by whichever gas or product shows up. The test is never separate from the reaction — the test *is* the reaction, read for its signature.

## II. BASE — the proton acceptor

### The distinction most students blur

**Base** is any proton acceptor. **Alkali** is a base that is *soluble in water*. Every alkali is a base; not every base is an alkali — insoluble bases like copper(II) hydroxide never touch water, but they still neutralize an acid on contact.

| Term | Relationship |
|---|---|
| Base | Accepts a proton — the general category |
| Alkali | A base that dissolves — the subset you can titrate directly |

### Common bases

| Base | Alkali? | Strength |
|---|---|---|
| NaOH | Yes | Strong |
| KOH | Yes | Strong |
| Ca(OH)2 | Slightly soluble — borderline | Strong (as far as it dissolves) |
| NH3 (ammonia solution) | Yes | Weak |
| Cu(OH)2, Fe(OH)3 | No | Insoluble, still basic |

### Operations

```
base + acid           → salt + water
alkali + ammonium salt, warm → salt + water + NH3 gas   (test: turns damp red litmus blue)
```

## III. SALT — the leftover

Salt was already given its relationship in Book 3: **what's left when acid gives and base takes and water walks off with the difference.** This section is where that leftover gets sorted.

### Multi-lens classification (expanded)

| Lens | Split | Logic |
|---|---|---|
| Composition | Normal / Acidic / Basic / Double / Complex | How completely the acid's H+ or base's OH– got replaced |
| Origin | Mineral / Organic | Same lens as acids, inherited |
| Solubility | Soluble / Insoluble | Decides preparation method — see below |
| Hydration | Anhydrous / Hydrated (carries water of crystallization) | Whether water molecules are locked into the crystal structure |

### Salt types, by composition

| Type | What happened | Example |
|---|---|---|
| Normal | All replaceable H+ (or OH–) fully exchanged | NaCl, from HCl + NaOH |
| Acidic | Not all H+ replaced — some acidic character remains | NaHSO4, NaHCO3 |
| Basic | Not all OH– replaced — some basic character remains | Basic lead carbonate |
| Double | Two different salts crystallizing together in fixed ratio | Potash alum, KAl(SO4)2·12H2O |
| Complex | A metal ion surrounded by ligands, the whole unit charged | [Cu(NH3)4]SO4 |

Relationship one-liner: **an acidic salt is an acid that never finished the neutralization — same unfinished-business logic as the radical that never finished leaving its ion.** The pattern from Book 3 (radical = unfinished ion) reappears here as (acidic salt = unfinished acid).

### Preparation method depends entirely on solubility — this is the part most students memorize as separate recipes instead of one decision tree

```
GATE → Is the target salt soluble?
   YES → titration (acid + alkali, indicator) or excess acid + insoluble base/carbonate/metal, then filter off excess and crystallize
   NO  → precipitation (double decomposition) — mix two soluble salts, insoluble salt drops out, filter
```

| Method | Used when | Example |
|---|---|---|
| Titration | Soluble salt from acid + soluble base | NaCl from HCl + NaOH |
| Excess solid + acid, filter, crystallize | Soluble salt from acid + insoluble base/carbonate/metal | CuSO4 from H2SO4 + excess CuO |
| Precipitation (double decomposition) | Insoluble salt | PbCl2 from Pb(NO3)2 + NaCl |
| Direct combination | Salt from elements directly | FeCl3 from Fe + Cl2 |

The entire method table collapses into one question: does the salt you want stay dissolved or fall out? Everything else is just which reactant is convenient to have in excess.

### Water of crystallization

```
hydrated salt --(heat)--> anhydrous salt + water (driven off as steam)
```

Blue copper(II) sulfate crystals (CuSO4·5H2O) turning white powder on heating is the visible proof: the blue color belongs to the water-bound structure, not the salt alone. Add water back to the white powder and the blue returns — this reversibility is the standard test for water.

### The Salt Scale — the axis pH doesn't show you

pH measures what's left dissolved in solution — it tells you the character of the medium. It never tells you what fell out of that medium, or what formed and walked away as a separate substance. That's a different axis entirely, and it runs vertically, straight through pH's neutral point at 7, rather than along it.

```
                    ESTERS
                 (condensation of organic acid + alcohol,
                  or lipid + glycerol — water released,
                  the organic answer to "salt")
                       ↑
                     SOAP
              (organic salt — saponification,
               ester + strong base → salt + glycerol)
                       ↑
                  COMPLEX SALT
              (metal ion + ligands, charged unit)
                       ↑
                   DOUBLE SALT
            (two salts crystallizing in fixed ratio)
                       ↑
                   BASIC SALT
              (some OH⁻ left unreplaced)
━━━━━━━━━━━━ NORMAL SALT ━━━━━━━━━━━━  ← sits on pH's neutral axis, the canonical case
              (acid fully + base fully → salt + water)
                       ↓
                   ACIDIC SALT
              (some H⁺ left unreplaced)
                       ↓
              MINERAL PRECIPITATE
        (insoluble inorganic salt, formed by
         double decomposition, not neutralization —
         BaSO4, AgCl, CaCO3)
                     BOTTOM
```

Read bottom to top: at the bottom sits the most mineral, least "chosen" salt — a precipitate that simply fell out of two solutions meeting, no acid donating a proton, no base accepting one, just insolubility doing the work (Section III's solubility rules, not neutralization at all). At the top sits the ester — not technically a salt by the strict acid+base definition, but built by the exact same *donor-meets-acceptor-releases-water* condensation logic, just with an alcohol standing in for the base and a fatty acid standing in for the mineral acid.

The normal salt in the middle is the only point on this scale that sits directly on pH's neutral axis — everything above and below it is still a "leftover of a giving-and-taking reaction," it's just increasingly organic going up, and increasingly mineral-raw going down.

Relationship one-liner: **pH tells you what's still in solution. The Salt Scale tells you what came out of it — and the more that "coming out" looks like biology (lipids, esters, fragrance) rather than geology (precipitates, minerals), the higher up this scale you climb.** Soap sits exactly where you'd expect, given Section VIII's soap equation — an organic salt, closer to the ester end than the mineral end, because saponification is one condensation step short of a true ester.

---

| Term | Behavior |
|---|---|
| Neutral | Neither donates nor accepts a proton — genuinely inert to both acid and base (pure water, table salt solution) |
| Amphoteric | Donates to a base, accepts from an acid — the same molecule switching roles depending on who it's reacting with |

```
amphoteric oxide/hydroxide + acid  → behaves as base → salt + water
amphoteric oxide/hydroxide + alkali → behaves as acid → salt + water
```

Al2O3 and Al(OH)3 are the standard examples — the same aluminum compound dissolves in both hydrochloric acid and sodium hydroxide, which is exactly why aluminum items shouldn't be cleaned with either strong acids or strong alkalis: it has no side it's safe on.

## V. HYDROLYSIS OF SALTS — the pH of a salt isn't automatically 7

A salt's dissolved pH depends on the strength of the acid and base that made it — and this is the operation most students skip straight past to memorize "salts are neutral," which is only true a quarter of the time.

| Acid strength | Base strength | Resulting salt solution pH |
|---|---|---|
| Strong | Strong | Neutral (NaCl — genuinely pH 7) |
| Strong | Weak | Acidic (NH4Cl — the weak base's conjugate can't hold ground) |
| Weak | Strong | Basic (CH3COONa — the weak acid's conjugate wins the tug of war) |
| Weak | Weak | Depends which is relatively stronger — rarely tested at this level |

Relationship one-liner: **a salt's pH is just a leftover scorecard — whichever parent (acid or base) was weaker loses its grip first, and the surviving stronger parent's character shows up in solution.**

## VI. QUALITATIVE ANALYSIS — identifying salts by operation, not guesswork

### Cation tests (add NaOH, then excess if needed)

| Cation | With NaOH (few drops) | With excess NaOH |
|---|---|---|
| Fe2+ | Green precipitate | Insoluble, stays green |
| Fe3+ | Reddish-brown precipitate | Insoluble, stays reddish-brown |
| Cu2+ | Blue precipitate | Insoluble, stays blue |
| Zn2+ | White precipitate | Dissolves — colorless solution (amphoteric) |
| Al3+ | White precipitate | Dissolves — colorless solution (amphoteric) |
| NH4+ | No precipitate | Warm gently → ammonia gas (turns damp red litmus blue) |

### Anion tests

| Anion | Test | Result |
|---|---|---|
| Carbonate (CO3^2-) | Add dilute acid | Effervescence, gas turns limewater cloudy |
| Sulfate (SO4^2-) | Add BaCl2 + dilute HCl | White precipitate (BaSO4), insoluble in acid |
| Chloride (Cl–) | Add AgNO3 + dilute HNO3 | White precipitate (AgCl) |
| Bromide (Br–) | Add AgNO3 + dilute HNO3 | Cream precipitate (AgBr) |
| Iodide (I–) | Add AgNO3 + dilute HNO3 | Yellow precipitate (AgI) |
| Nitrate (NO3–) | Brown ring test (Fe2+ + concentrated H2SO4, layered) | Brown ring at the interface |

The pattern worth naming: **most qualitative tests are just the solubility rules from Section III, run in reverse — you're not discovering new chemistry, you're using a known insolubility to fish out an unknown ion.**

## VII. COMPLEX / ALLOY / POLYMER — the fifth branch

This branch doesn't form by acid-base exchange at all — it forms by molecules or ions simply agreeing to associate, which is why it sits apart from acid/base/salt/neutral as its own category.

### Coordination compounds

```
complex ion = central metal ion + ligands (donors that share a lone pair, not a full electron)
```

| Term | Relationship |
|---|---|
| Ligand | A molecule or ion that lends electrons without fully bonding the way a covalent partner would — a guest, not a spouse |
| Coordination number | How many ligands surround the central ion — usually 4 or 6 |
| Complex salt | A complex ion paired with a counter-ion to balance charge — [Cu(NH3)4]SO4 |

### Alloys — a compound that isn't really a compound

An alloy is a mixture, not a compound in the strict bonding sense — metal atoms of different sizes packed into one lattice, no fixed formula, no stoichiometry.

| Alloy | Composition | Why it's better than the pure metal |
|---|---|---|
| Steel | Iron + carbon | Carbon atoms jam the iron lattice, blocking the layers that let pure iron bend easily — harder, less malleable |
| Brass | Copper + zinc | Same lattice-jamming logic — harder than either pure metal |
| Duralumin | Aluminum + copper | Aluminum's lightness, steel-adjacent strength |

Relationship one-liner: **an alloy gets stronger for the same reason a branched hydrocarbon boils lower — different-sized parts disrupt a pattern that pure, uniform packing would otherwise allow.** The lattice-jamming logic and the branching logic from Book 3 are the same physical idea, wearing two different course names.

### Polymers, revisited

Already covered in Organic (Section V) as addition/condensation. Worth restating here only for placement: a polymer belongs in this branch because, like a complex, it is molecules associating in bulk rather than acid and base exchanging protons.

## VIII. SEPARATION TECHNIQUES, ANALYSIS & PRACTICAL SAFETY

Every separation technique exploits exactly one physical difference between the things being separated. Learn the property, and the technique stops being a memorized list and becomes a lookup.

### The Gate

```
GATE → What property differs between the components?
   Particle size (dissolved vs undissolved) → Filtration
   Boiling point, same phase mixture         → Distillation (simple: 1 liquid + dissolved solid; fractional: 2+ liquids, close boiling points)
   Solubility at different temperatures       → Crystallization
   Density, immiscible liquids                → Separating funnel (decantation)
   Solubility in a moving solvent (mobility)  → Chromatography
   Solid → gas directly, no liquid phase      → Sublimation
```

| Technique | Property exploited | Example |
|---|---|---|
| Filtration | Particle size | Sand from water |
| Simple distillation | Boiling point, solid vs liquid | Salt from water |
| Fractional distillation | Boiling point, liquid vs liquid | Ethanol from water; crude oil (Organic Section VI) |
| Crystallization | Solubility drops as temperature drops | Copper sulfate crystals from solution |
| Separating funnel | Density, immiscibility | Oil from water |
| Chromatography | Differential solvent mobility | Separating dye/ink components |
| Sublimation | Direct solid-to-gas transition | Iodine, ammonium chloride from a mixture |

Relationship one-liner: **there is no "separation chapter" distinct from everything before it — each technique is just Section III's solubility gate or Section I's boiling-point trend, turned into a hands-on procedure.**

---

### Solubility rules — the table every salt decision above depended on

This is the table Section III's preparation method gate assumes you already have memorized. State it once, properly, here:

| Ion / salt family | Solubility | Exceptions (insoluble) |
|---|---|---|
| Group 1 (Na+, K+) and ammonium (NH4+) salts | Always soluble | None |
| Nitrates (NO3–) | Always soluble | None |
| Chlorides, bromides, iodides | Generally soluble | AgCl, PbCl2, Hg2Cl2 (and their Br/I equivalents) — insoluble |
| Sulfates (SO4^2-) | Generally soluble | BaSO4, PbSO4, CaSO4 (slightly) — insoluble |
| Carbonates (CO3^2-) | Generally insoluble | Group 1 carbonates and ammonium carbonate — soluble |
| Hydroxides (OH–) | Generally insoluble | Group 1 hydroxides — soluble; Ca(OH)2 — slightly soluble |

Read this table before choosing a preparation method, not after: **if both your target salt and its likely double-decomposition partner are soluble, titration is your route; if the salt itself sits in an "exceptions" column, precipitation is the only route that will ever work.**

---

### Titration — matching indicator to the acid-base pair, not guessing

An indicator only works if it changes color *at* the pH the equivalence point actually sits at — using the wrong one gives a false endpoint even though the color change looks convincing.

| Acid + Base pair | Equivalence point pH | Correct indicator |
|---|---|---|
| Strong acid + Strong base | ~7 (neutral) | Either phenolphthalein or methyl orange |
| Strong acid + Weak base | Acidic (below 7) | Methyl orange only |
| Weak acid + Strong base | Basic (above 7) | Phenolphthalein only |
| Weak acid + Weak base | No sharp endpoint | Avoid titrating this pair — no indicator gives a clean read |

This table is Section V's hydrolysis logic, applied forward instead of backward: there, you predicted a salt's pH from its parent acid/base; here, you're choosing the indicator that will actually catch the color change at that same predicted pH.

### What's safe to react, and what isn't

```
RULE 1: Always add acid to water, never water to concentrated acid.
        (Diluting concentrated acid releases heat suddenly — adding water to acid lets a small
        amount of water boil instantly at the surface and spit acid back out. Adding acid to
        a larger volume of water spreads the heat instead of concentrating it at one point.)

RULE 2: Never mix bleach (hypochlorite) with acid — releases toxic chlorine gas.

RULE 3: Never heat a sealed container. Gas produced by any reaction needs an escape route,
        or the container becomes the explosion.

RULE 4: Test for gas identity before assuming it — a squeaky pop (H2), a glowing splint
        relighting (O2), and limewater turning cloudy (CO2) are different gases with
        different further handling needs, not interchangeable "some gas came off" results.
```

Relationship one-liner: **most lab safety rules aren't arbitrary caution — they're the same reaction operations from Sections I, II, and III, except run at a scale or speed where the byproduct (heat, gas, pressure) stops being a tidy observation and starts being a hazard.**

## THE COMPLETE HIERARCHY, RESTATED

```
MOLECULE
   ↓
OXIDE (the unfinished sentence — which branch is this element about to join?)
   ↓
GATE → ACID / BASE / SALT / NEUTRAL-AMPHOTERIC / COMPLEX-ALLOY-POLYMER
   ↓ (acid)                ↓ (base)              ↓ (salt)                    ↓ (neutral/amphoteric)          ↓ (complex/alloy)
strength, basicity,    base vs alkali,      composition, solubility,    genuinely inert vs switches    ligand/coordination number,
origin, oxygen content  strength             hydration                   sides depending on partner      lattice-jamming in alloys
   ↓                        ↓                     ↓                             ↓                                ↓
reaction operations     reaction operations   preparation method gate,     amphoteric operations           complex salts, alloys,
(4 signature tests)                          hydrolysis pH table,                                          polymers (cross-reference
                                              qualitative analysis                                          to Organic Section V)
```

Compound chemistry is not five topics. It is one fork, five directions, and every "new" reaction inside each direction is the same acid-base-salt logic, asked one more time with a different partner in the room.

Separation, solubility, and lab safety are not a sixth topic bolted on at the end — they are the same fork, made physical: filtration and crystallization are the solubility gate with your hands in it, titration indicator choice is the hydrolysis table read forward, and every safety rule is a reaction operation running at a scale where the byproduct stops being an observation and starts being a hazard.

## BEFORE YOU CLOSE THIS CHAPTER

Not a summary. A checkpoint. If any of these five stall you, that's the pocket to reopen — not the whole chapter.

1. A metal oxide and a nonmetal oxide both fall into water. Without naming either compound, what family does each solution belong to, and why did the element's identity alone already decide it?
2. You're handed a white solid and told only that it's a salt. Name the two lab results that alone would tell you whether to prepare more of it by titration or by precipitation.
3. A salt tests neutral in solution. State the one thing you now know for certain about the acid and base that made it — and the one thing you still don't know.
4. Bromine water decolorizes a hydrocarbon sample. Which bond broke to cause that, and what does the same bond's absence in benzene tell you about why aromatic compounds resist this exact test?
5. Someone hands you concentrated sulfuric acid and a beaker of water and asks which goes into which first. Answer in one sentence, and in a second sentence say what happens if you get it backward.

If question five is the one you hesitated on: that's not a memory gap. That's the one question in this chapter where getting it wrong doesn't cost you a mark — it costs you the top layer of skin on your hand. Some pockets in this book are inconvenient to forget. That one isn't allowed to be.
