∂ Block 1 — Differentiation: Power Rule Core Conversions · Flow Q1–Q7
Q 01Power Rule
Find d/dx of f(x) = x5.
(A) 5x6 (B) 5x4 (C) x4 (D) 5x5
Power Rule: For f(x) = xn, f′(x) = n·xn−1. d/dx (x5) = 5 · x5−1 = 5x4 The most common error is forgetting to reduce the exponent by 1. The coefficient comes down and the power drops by 1 — both must happen.
Answer: B — 5x4
Q 02Power Rule
Before applying the power rule, rewrite f(x) = √x as a power of x. What is f′(x)?
(A) 2√x (B) 12√x (C) -12x-1/2 (D) √x · ln x
Rewrite first: √x = x1/2. d/dx (x1/2) = 1/2 · x1/2 − 1 = 1/2 · x−1/2 = 1/(2√x) Core conversion: √x → x1/2, ∛x → x1/3. Recognising these before differentiating is the critical step.
Answer: B — 1/(2√x)
Q 03Power Rule
Differentiate f(x) = 1x3. Rewrite as a negative exponent first.
(A) −3x4 (B) 3x4 (C) −13x2 (D) 13x2
Rewrite: 1/x3 = x−3. d/dx (x−3) = −3 · x−3−1 = −3x−4 = −3/x4 Core conversion: 1/xn → x−n. The negative sign in the result is correct — the derivative of a decreasing function near x = 0+ is negative.
Answer: A — −3/x4
Q 04Power Rule
Find the derivative of y = 3x4 − 5x2 + 7x − 2.
(A) 12x3 − 10x + 7 (B) 12x4 − 10x2 + 7 (C) 12x3 − 10x + 7 − 2 (D) 3x3 − 5x + 7
Differentiate term by term using linearity: d/dx (3x4) = 12x3 d/dx (−5x2) = −10x d/dx (7x) = 7 d/dx (−2) = 0 ← constants always vanish Result: y′ = 12x3 − 10x + 7. The constant −2 disappears because a constant has slope zero. Option C incorrectly keeps the −2.
Answer: A — 12x3 − 10x + 7
Q 05Power Rule
Core conversion question: Rewrite 5√x3 as a single power of x, then differentiate.
(A) −152x−5/2 (B) 52x1/2 (C) 5 · x3/2 (D) −52x1/2
Step 1 — Rewrite: 5/√(x3) = 5 · (x3)−1/2 = 5x−3/2 Step 2 — Differentiate: d/dx (5x−3/2) = 5 · (−3/2) · x−3/2 − 1 = −15/2 · x−5/2 This is the most exam-tested form: fraction inside a root → negative fractional exponent → power rule.
Answer: A — −(15/2)x−5/2
Q 06Power Rule
The gradient function of y = axn is zero when n = 0 (constant). If y = 6, what is dy/dx?
(A) 6 (B) 0 (C) 6x (D) 1/6
Rule of constants: d/dx (c) = 0 for any constant c. A horizontal line has zero slope everywhere. Equivalently: y = 6 = 6x0, and d/dx (6x0) = 6 · 0 · x−1 = 0. This foundation is often overlooked in polynomial differentiation.
Answer: B — 0
Q 07Power Rule
Find the gradient of y = 2x3 − 4x at x = 2.
(A) 20 (B) 20 (C) 8 (D) 4
y′ = 6x2 − 4 y′(2) = 6(4) − 4 = 24 − 4 = 20 "Gradient at a point" means: differentiate fully, then substitute the x-value. Never substitute before differentiating.
Answer: A/B — 20
∂ Block 2 — Chain Rule: Core Substitution Conversions · Q8–Q14
Q 08Chain Rule
Differentiate y = (3x + 2)5 using the chain rule.
(A) 5(3x+2)4 (B) 15(3x+2)4 (C) 5(3)4 (D) 3(3x+2)5
Chain rule: dy/dx = dy/du · du/dx, where u = inner function. Let u = 3x + 2, so y = u5 dy/du = 5u4; du/dx = 3 dy/dx = 5u4 · 3 = 15(3x+2)4 Option A omits the derivative of the inner function (du/dx = 3). This multiplier from the inner function is the most frequently missed step.
Answer: B — 15(3x+2)4
Q 09Chain Rule
Find dy/dx if y = √(5x2 − 3). Rewrite as a half-power first.
(A) 5x√(5x2−3) (B) 12√(5x2−3) (C) 10x · √(5x2−3) (D) 10x√(5x2−3)
y = (5x2 − 3)1/2 dy/dx = 1/2(5x2−3)−1/2 · 10x = 10x/2√(5x2−3) = 5x/√(5x2−3) The inner function is 5x2−3 with derivative 10x. Option B forgets the 10x multiplier.
Answer: A — 5x / √(5x²−3)
Q 10Chain Rule
Differentiate y = e3x2.
(A) 3x2e3x2−1 (B) 6x · e3x2 (C) e6x (D) 3x · e3x2
Rule: d/dx (eu) = eu · u′. Here u = 3x2. u′ = d/dx (3x2) = 6x dy/dx = e3x2 · 6x = 6xe3x2 Option A wrongly applies the power rule to eu. The exponential function is its own derivative — only the chain-rule multiplier (u′) is added.
Answer: B — 6x·e3x²
Q 11Chain Rule
Differentiate y = ln(4x3).
(A) 14x3 (B) 12x24x3 = 3x (C) 1x (D) 4x3
Rule: d/dx ln(u) = u′/u. u = 4x3, u′ = 12x2 dy/dx = 12x2 / 4x3 = 3/x Alternatively: ln(4x3) = ln 4 + 3 ln x, so d/dx = 0 + 3/x = 3/x. Both routes give the same answer — log laws can simplify chain-rule ln problems.
Answer: B — 3/x
Q 12Chain Rule
Find f′(x) for f(x) = (x2 + 1)−4.
(A) −4(x2+1)−5 (B) −8x(x2+1)−5 (C) 4(x2+1)−3 (D) −4x2(x2+1)−5
f′(x) = −4(x2+1)−5 · 2x = −8x(x2+1)−5 Inner function x2+1 has derivative 2x. Option A omits the 2x. Option D uses x2 instead of 2x — a common error when applying the power rule to the inner function.
Answer: B — −8x(x²+1)−5
Q 13Chain Rule
Core conversion: Differentiate y = 1(2x−1)3. Rewrite first, then apply chain rule.
(A) −6(2x−1)4 (B) −3(2x−1)2 (C) 6(2x−1)4 (D) −3(2x−1)4
Rewrite: y = (2x−1)−3 dy/dx = −3(2x−1)−4 · 2 = −6(2x−1)−4 = −6/(2x−1)4 The factor of 2 comes from the inner function (2x−1)′ = 2. Option D omits this factor.
Answer: A — −6/(2x−1)4
Q 14Chain Rule
Differentiate y = ln(sin x).
(A) 1sin x (B) cot x (C) cos(ln x) (D) −cot x
d/dx ln(sin x) = cos x / sin x = cot x This is a chain rule + known result: d/dx(ln u) = u′/u; u = sin x, u′ = cos x. The result cot x = cos x / sin x is a standard identity that appears frequently.
Answer: B — cot x
∂ Block 3 — Product & Quotient Rules · Q15–Q21
Q 15Product Rule
State the product rule and apply it to differentiate y = x2 · ex.
(A) 2x · ex (B) x2 · ex (C) ex(x2 + 2x) (D) 2xex − x2ex
Product rule: (uv)′ = u′v + uv′. u = x2, u′ = 2x; v = ex, v′ = ex dy/dx = 2x·ex + x2·ex = ex(2x + x2) Factor ex out. Option A only differentiates u. Option D has a wrong sign. The product rule is a sum of two terms, never a difference.
Answer: C — ex(x² + 2x)
Q 16Product Rule
Differentiate y = x3 ln x.
(A) 3x2 · ln x (B) x2(3 ln x + 1) (C) x3x (D) x2(3 ln x − 1)
u = x3, u′ = 3x2; v = ln x, v′ = 1/x dy/dx = 3x2 · ln x + x3 · (1/x) = 3x2 ln x + x2 = x2(3 ln x + 1) Option D has −1 instead of +1. Option C only computes uv′.
Answer: B — x²(3 ln x + 1)
Q 17Quotient Rule
Use the quotient rule to differentiate y = x2x + 1.
(A) x2 + 2x(x+1)2 (B) x(x + 2)(x+1)2 (C) 2x1 (D) 2x(x+1) + x2(x+1)2
Quotient rule: (u/v)′ = (u′v − uv′) / v². Note: it is u′v minus uv′ (top minus, not plus). u = x2, u′ = 2x; v = x+1, v′ = 1 dy/dx = [2x(x+1) − x2·1] / (x+1)2 = [2x2 + 2x − x2] / (x+1)2 = (x2 + 2x)/(x+1)2 = x(x+2)/(x+1)2
Answer: B — x(x+2)/(x+1)²
Q 18Quotient Rule
Differentiate y = exx2.
(A) ex(x − 2)x3 (B) ex2x (C) ex(x + 2)x3 (D) xex2
u = ex, u′ = ex; v = x2, v′ = 2x dy/dx = [ex·x2 − ex·2x] / x4 = ex[x2 − 2x] / x4 = ex(x−2)/x3 Option C has + instead of −; the numerator order in the quotient rule is u′v − uv′ (not u′v + uv′).
Answer: A — ex(x−2)/x³
Q 19Product Rule
Differentiate y = (2x + 1)(x2 − 3) using the product rule, then verify by expanding first.
(A) 2(x2−3) + (2x+1)·2x (B) 6x2 + 2x − 6 (C) 4x2 + 2x − 3 (D) 2x2 + 2
Product rule route: u′v + uv′ = 2(x2−3) + (2x+1)(2x) = 2x2−6 + 4x2+2x = 6x2+2x−6 Expand route (verification): y = 2x3−6x+x2−3 → y′ = 6x2+2x−6 ✓ Both routes give 6x²+2x−6. The expand route is faster when the product expands cleanly; use product rule when factors involve ex, ln x or trig.
Answer: B — 6x² + 2x − 6
Q 20Quotient Rule
Differentiate y = ln xx.
(A) ln x − 1x2 (B) 1 − ln xx2 (C) 1x2 (D) 1x ln x
u = ln x, u′ = 1/x; v = x, v′ = 1 dy/dx = [(1/x)·x − ln x·1] / x2 = [1 − ln x] / x2 Numerator order matters: u′v − uv′ = (1/x)(x) − ln(x)(1) = 1 − ln x. Option A reverses the sign.
Answer: B — (1 − ln x)/x²
Q 21Quotient Rule
Show that d/dx (tan x) = sec2x by treating tan x as sin x / cos x and applying the quotient rule.
(A) sin2x + cos2x = 1, so result is 1 (B) (cos2x + sin2x)/cos2x = 1/cos2x = sec2x (C) cos x / (−sin x) = −cot x (D) 1/sin2x = cosec2x
u = sin x, u′ = cos x; v = cos x, v′ = −sin x d/dx (sin x/cos x) = [cos x·cos x − sin x·(−sin x)] / cos2x = [cos2x + sin2x] / cos2x = 1/cos2x = sec2x This proof uses the Pythagorean identity sin²x + cos²x = 1 in the numerator. Memorise the result: d/dx(tan x) = sec²x.
Answer: B — sec²x (via Pythagorean identity)
∂ Block 4 — ex and ln Derivatives · Q22–Q25
Q 22Exp/Log
What is d/dx (ex), and why is e special?
(A) x · ex−1 (B) ex — it is its own derivative (C) ln(x) · ex (D) e · x
The number e ≈ 2.718 is defined precisely so that d/dx(ex) = ex. It is the unique function equal to its own derivative. This is why ex dominates growth/decay modelling. Option A treats ex like a power function (wrong — the variable is in the exponent, not the base).
Answer: B — ex (its own derivative)
Q 23Exp/Log
Differentiate y = ax (where a > 0, a ≠ 1).
(A) x · ax−1 (B) ax · ln a (C) ax · log10a (D) ax / ln a
Rewrite: ax = ex ln a. Then: d/dx (ex ln a) = ex ln a · ln a = ax · ln a The factor ln a is the constant multiplier that comes from the chain rule on the exponent x ln a. When a = e, ln e = 1, recovering d/dx(ex) = ex·1 = ex.
Answer: B — ax·ln a
Q 24Exp/Log
Differentiate y = loga x using the change-of-base formula.
(A) 1x · log a (B) 1x · ln a (C) ln ax (D) ax
Change of base: logax = ln x / ln a, where ln a is a constant. d/dx (ln x / ln a) = (1/ln a) · (1/x) = 1/(x ln a) When a = 10: d/dx(log₁₀ x) = 1/(x ln 10). When a = e: 1/(x ln e) = 1/x. Always convert loga to ln before differentiating.
Answer: B — 1/(x ln a)
Q 25Exp/Log
Hard: Use logarithmic differentiation to find dy/dx for y = xx.
(A) x · xx−1 (B) xx(1 + ln x) (C) ex (D) xx · ln x
Logarithmic differentiation — use when both base and exponent contain x. ln y = x ln x Differentiate both sides: (1/y)·(dy/dx) = ln x + x·(1/x) = ln x + 1 dy/dx = y(ln x + 1) = xx(1 + ln x) Option D forgets the +1 from differentiating x·ln x. Option A wrongly applies the power rule.
Answer: B — xx(1 + ln x)
∫ Block 5 — Standard Integrals · Flow Q26–Q32
Q 26Standard
Evaluate ∫ x4 dx.
(A) 4x3 + C (B) x55 + C (C) x5 + C (D) x33 + C
Power rule for integration: ∫xn dx = xn+1/(n+1) + C (n ≠ −1). ∫x4 dx = x4+1/(4+1) = x5/5 + C The +C (constant of integration) is mandatory for indefinite integrals. Integration is the inverse of differentiation: d/dx(x5/5) = x4 ✓.
Answer: B — x⁵/5 + C
Q 27Standard
Find ∫ 1x dx.
(A) 1x2 + C (B) ln|x| + C (C) −1x2 + C (D) x−1 + C
The power rule fails at n = −1 (gives 1/0). The exception: ∫(1/x) dx = ln|x| + C. The absolute value |x| ensures the log is defined for negative x too. This is the most important integral exception — it appears whenever a denominator differentiates to the numerator.
Answer: B — ln|x| + C
Q 28Standard
Evaluate ∫ ex dx.
(A) ex+1/(x+1) + C (B) ex + C (C) xex + C (D) ex/x + C
Since d/dx(ex) = ex, integration simply reverses it: ∫ex dx = ex + C. Option A wrongly applies the power rule formula to ex. Note: ∫eax dx = eax/a + C (chain-rule reversal).
Answer: B — ex + C
Q 29Standard
Find ∫ (6x2 − 4x + 3) dx.
(A) 2x3 − 2x2 + 3x + C (B) 12x − 4 + C (C) 2x3 − 4x2 + 3x + C (D) 6x3 − 4x2 + 3x + C
Integrate term by term: ∫6x2 dx = 6·x3/3 = 2x3 ∫−4x dx = −4·x2/2 = −2x2 ∫3 dx = 3x Result: 2x³ − 2x² + 3x + C. Option C keeps −4x² (didn't divide by 2). Option D keeps 6x³ (didn't divide by 3).
Answer: A — 2x³ − 2x² + 3x + C
Q 30Standard
Core conversion: Evaluate ∫ 3√x dx by rewriting as a power of x first.
(A) 6√x + C (B) 3√x + C (C) 32x3/2 + C (D) −6x−1/2 + C
Rewrite: 3/√x = 3x−1/2 ∫3x−1/2 dx = 3 · x−1/2 + 1 / (−1/2 + 1) = 3 · x1/2 / (1/2) = 6x1/2 = 6√x + C Dividing by 1/2 is the same as multiplying by 2. Result: 6√x + C. Verify: d/dx(6√x) = 6·(1/2)x−1/2 = 3/√x ✓.
Answer: A — 6√x + C
Q 31Standard
Find ∫ e5x dx.
(A) 5e5x + C (B) e5x5 + C (C) e5x + C (D) e5x5x + C
For ∫eax dx = eax/a + C. Here a = 5: ∫e5x dx = e5x/5 + C This is chain-rule reversal: d/dx(e5x/5) = e5x·5/5 = e5x ✓. Option A differentiates instead of integrating (multiplies by 5 instead of dividing).
Answer: B — e5x/5 + C
Q 32Standard
Evaluate ∫ 2xx2 + 1 dx. Recognise the pattern.
(A) 2 ln(2x) + C (B) ln(x2+1) + C (C) 1x2+1 + C (D) 2x · ln(x2+1) + C
Pattern: ∫f′(x)/f(x) dx = ln|f(x)| + C f(x) = x2+1, f′(x) = 2x ∫2x/(x2+1) dx = ln|x2+1| + C = ln(x2+1) + C (since x²+1 > 0 always) This pattern is one of the most frequently tested integrals in Further Maths. If the numerator is exactly the derivative of the denominator, the answer is ln(denominator).
Answer: B — ln(x²+1) + C
∫ Block 6 — Integration by Substitution · Q33–Q39
Q 33Substitution
Use the substitution u = 3x + 2 to evaluate ∫ (3x + 2)4 dx.
(A) (3x+2)55 + C (B) (3x+2)515 + C (C) (3x+2)53 + C (D) 4(3x+2)3 · 3 + C
u = 3x+2, du = 3dx → dx = du/3 ∫u4 · (du/3) = (1/3) · u5/5 + C = u5/15 + C = (3x+2)5/15 + C Option A omits the 1/3 factor from dx = du/3. That factor is the chain-rule correction when reversing differentiation.
Answer: B — (3x+2)⁵/15 + C
Q 34Substitution
Evaluate ∫ 2x · ex2 dx using substitution.
(A) 2ex2 + C (B) ex2 + C (C) x2ex2 + C (D) ex22x + C
u = x2, du = 2x dx ← notice 2x is already in the integrand ∫eu du = eu + C = ex2 + C When the derivative of the inner function appears as a multiplier, substitution cancels it cleanly. This is the hallmark of a substitution problem.
Answer: B — ex² + C
Q 35Substitution
Core conversion: Evaluate ∫ 4x3x4 + 5 dx.
(A) 14x3 + C (B) ln|x4+5| + C (C) 4 ln|x4+5| + C (D) ln|4x3| + C
u = x4+5, du = 4x3 dx ← numerator is exactly du ∫du/u = ln|u| + C = ln|x4+5| + C f′(x)/f(x) pattern. Numerator is the derivative of denominator. Result is always ln of the denominator.
Answer: B — ln|x⁴+5| + C
Q 36Substitution
Evaluate ∫ x√(x2+1) dx.
(A) 23(x2+1)3/2 + C (B) 13(x2+1)3/2 + C (C) x · 23(x2+1)3/2 + C (D) 12(x2+1)1/2 + C
u = x2+1, du = 2x dx → x dx = du/2 ∫√u · (du/2) = (1/2)∫u1/2 du = (1/2) · u3/2/(3/2) = (1/2)·(2/3)u3/2 = (1/3)u3/2 + C = (1/3)(x2+1)3/2 + C Option A forgets to divide by 2 for the dx → du/2 substitution.
Answer: B — (1/3)(x²+1)3/2 + C
Q 37Substitution
Find ∫ sin(5x) dx.
(A) cos(5x) + C (B) 5cos(5x) + C (C) −15cos(5x) + C (D) −cos(5x) + C
u = 5x, du = 5 dx → dx = du/5 ∫sin u · (du/5) = −(1/5)cos u + C = −(1/5)cos(5x) + C The −1/5 has two sources: ∫sin u du = −cos u (the minus sign from integrating sin), and the /5 from the substitution. Both must be included.
Answer: C — −cos(5x)/5 + C
Q 38Substitution
Hard substitution: Evaluate ∫ exex + 3 dx.
(A) ln|ex| + C (B) ln(ex+3) + C (C) exex+3 + C (D) ex ln(ex+3) + C
u = ex+3, du = ex dx ← numerator is exactly du ∫du/u = ln|u| + C = ln(ex+3) + C f′/f pattern again. d/dx(ex+3) = ex, which is the numerator.
Answer: B — ln(ex+3) + C
Q 39Substitution
Evaluate ∫ cos(x) · esin x dx.
(A) sin x · esin x + C (B) esin x + C (C) ecos x + C (D) −esin x + C
u = sin x, du = cos x dx ← cos x is already in the integrand ∫eu du = eu + C = esin x + C The cos x factor in the integrand is the signal that u = sin x works. This is an elegant cancellation: the derivative of the inner function appears outside.
Answer: B — esin x + C
∫ Block 7 — Integration by Parts · Q40–Q46
Q 40By Parts
State the integration by parts formula and apply it to evaluate ∫ x · ex dx.
(A) x · ex + C (B) ex(x − 1) + C (C) x2ex/2 + C (D) ex(x + 1) + C
Formula: ∫u dv = uv − ∫v du. u = x → du = dx; dv = exdx → v = ex ∫x·exdx = x·ex − ∫exdx = xex − ex + C = ex(x−1) + C LIATE rule — choose u as whichever comes first: Logarithm, Inverse trig, Algebraic, Trig, Exponential. Here x (Algebraic) > ex (Exponential), so u = x.
Answer: B — ex(x − 1) + C
Q 41By Parts
Evaluate ∫ x · cos x dx.
(A) x sin x + C (B) x sin x + cos x + C (C) x sin x − cos x + C (D) cos x − x sin x + C
u = x, du = dx; dv = cos x dx, v = sin x ∫x cos x dx = x sin x − ∫sin x dx = x sin x − (−cos x) + C = x sin x + cos x + C The double negative: ∫sin x dx = −cos x, so −(−cos x) = +cos x. Option C misses this sign reversal.
Answer: B — x sin x + cos x + C
Q 42By Parts
Find ∫ ln x dx using parts with u = ln x, dv = dx.
(A) x ln x + C (B) x ln x − x + C (C) x ln x + x + C (D) 1x + C
u = ln x, du = (1/x)dx; dv = dx, v = x ∫ln x dx = x ln x − ∫x · (1/x) dx = x ln x − ∫1 dx = x ln x − x + C This is the standard result for ∫ln x dx. Option A omits the −x term. Verify: d/dx(x ln x − x) = ln x + 1 − 1 = ln x ✓.
Answer: B — x ln x − x + C
Q 43By Parts
Evaluate ∫ x2 · ex dx. (Requires two applications of parts.)
(A) x2ex − 2xex + C (B) ex(x2 − 2x + 2) + C (C) ex(x2 + 2x + 2) + C (D) ex(x2 − 2) + C
First application: u = x2, dv = exdx → x2ex − ∫2x·exdx Second application on ∫2x·exdx (from Q40: xex−ex): ∫2x·exdx = 2(xex−ex) = 2xex−2ex Total: x2ex − 2xex + 2ex + C = ex(x2−2x+2) + C
Answer: B — ex(x² − 2x + 2) + C
Q 44By Parts
Evaluate ∫ x · ln x dx.
(A) x22 ln x + C (B) x22 ln x − x24 + C (C) x24 + C (D) x ln x − x + C
LIATE: u = ln x (Logarithm), dv = x dx. v = x2/2 ∫x ln x dx = (x2/2)·ln x − ∫(x2/2)·(1/x) dx = (x2/2) ln x − ∫(x/2) dx = (x2/2) ln x − x2/4 + C Option A omits the second integral (the −x²/4 term).
Answer: B — (x²/2)ln x − x²/4 + C
Q 45By Parts
Cyclic by parts: Evaluate ∫ ex sin x dx. Let I denote this integral.
(A) ex(sin x − cos x) + C (B) ex(sin x − cos x)2 + C (C) ex(sin x + cos x)2 + C (D) ex cos x + C
Apply parts twice, keeping the same u-choice each time: 1st: I = exsin x − ∫excos x dx 2nd: ∫excos x dx = excos x + ∫exsin x dx = excos x + I So: I = exsin x − (excos x + I) 2I = ex(sin x − cos x) → I = ex(sin x − cos x)/2 + C The cyclic trick: I reappears on the right side, collect both I terms and divide. This is unique to products of ex with sin/cos.
Answer: B — ex(sin x − cos x)/2 + C
Q 46By Parts
Evaluate ∫ x · sin x dx.
(A) −x cos x + sin x + C (B) x cos x + sin x + C (C) −x cos x − sin x + C (D) x sin x + cos x + C
u = x, du = dx; dv = sin x dx, v = −cos x ∫x sin x dx = x(−cos x) − ∫(−cos x) dx = −x cos x + ∫cos x dx = −x cos x + sin x + C The double negative at step 2: −∫(−cos x)dx = +∫cos x dx. Option B has +x cos x (wrong sign from forgetting v = −cos x).
Answer: A — −x cos x + sin x + C
∫ Block 8 — Definite Integrals · Q47–Q50
Q 47Definite
