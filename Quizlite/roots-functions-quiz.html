🔹 Topic 1: Evaluating f(x) — Quadratic Expressions (Direct Q1–Q7)
Core trap: "find f(2)" means replace every x with 2, then compute — it does NOT mean "2 times f" or "f plus 2".
QUESTION 1
Given \(f(x) = x^2 - 5x + 6\), find \(f(2)\).
(A) 0(B) 4(C) 8(D) −4
Solution: Replace every \(x\) with 2: \(f(2) = (2)^2 - 5(2) + 6 = 4 - 10 + 6 = 0\).
Trap: do not compute \(2f\) or \(f+2\). The bracket notation means substitution.
Answer: A
QUESTION 2
If \(f(x) = 2x^2 + 3x - 5\), evaluate \(f(-1)\).
(A) −6(B) 0(C) −10(D) 4
Solution: \(f(-1) = 2(-1)^2 + 3(-1) - 5 = 2(1) - 3 - 5 = 2 - 8 = -6\).
Trap: \((-1)^2 = +1\), not \(-1\). Squaring a negative always gives positive.
Answer: A
QUESTION 3
If \(g(x) = x^2 - 4\), find \(g(0)\).
(A) −4(B) 0(C) 4(D) 2
Solution: \(g(0) = 0^2 - 4 = 0 - 4 = -4\).
This result is also the y-intercept of the parabola — notice that.
Answer: A
QUESTION 4
Given \(p(x) = x^2 - 3x + 2\), find \(p(3)\).
(A) 2(B) 5(C) 0(D) −2
Solution: \(p(3) = 9 - 9 + 2 = 2\).
Since \(p(3) \neq 0\), then \(x = 3\) is NOT a root of \(p(x)\).
Answer: A
QUESTION 5
Given \(f(x) = x^2 - 3x + 2\), which value of \(x\) gives \(f(x) = 0\)?
(A) \(x = 1\) or \(x = 2\)(B) \(x = -1\) or \(x = -2\) (C) \(x = 1\) only(D) \(x = 3\)
Solution: Set \(x^2 - 3x + 2 = 0\). Factorise: \((x-1)(x-2) = 0\), so \(x = 1\) or \(x = 2\).
These values are the ZEROS of f and the ROOTS of the equation \(f(x)=0\). The two words mean the same thing.
Answer: A
QUESTION 6
If \(f(x) = x^2 - 4\), find \(f(a + 1)\) in simplified form.
(A) \(a^2 + 2a - 3\)(B) \(a^2 - 3\) (C) \(a^2 + 2a + 1\)(D) \(a^2 - 4\)
Solution: Replace \(x\) with \((a+1)\): \(f(a+1) = (a+1)^2 - 4 = a^2 + 2a + 1 - 4 = a^2 + 2a - 3\).
Expand the bracket fully before collecting terms. \((a+1)^2 \neq a^2 + 1\).
Answer: A
QUESTION 7
Given \(h(x) = 3x^2 - x + 4\), find \(h(0) + h(1)\).
(A) 10(B) 6(C) 4(D) 14
Solution: \(h(0) = 0 - 0 + 4 = 4\). \(h(1) = 3 - 1 + 4 = 6\). Sum \(= 4 + 6 = 10\).
Evaluate each separately, then add. Do not compute \(h(0+1) = h(1)\) — that is a different question.
Answer: A
🔹 Topic 2: Roots by Factorisation (Direct Q8–Q14)
Core trap: if the factor is (x − 3), the root is x = +3, NOT −3. The factor sets the expression to zero, so x − 3 = 0 ⟹ x = 3.
QUESTION 8
Solve \(x^2 - 7x + 12 = 0\) by factorisation.
(A) \(x = 3\) or \(x = 4\)(B) \(x = -3\) or \(x = -4\) (C) \(x = 3\) or \(x = -4\)(D) \(x = 6\) or \(x = 2\)
Solution: Need two numbers that multiply to \(+12\) and add to \(-7\): that is \(-3\) and \(-4\).
\((x - 3)(x - 4) = 0 \Rightarrow x = 3\) or \(x = 4\).
The factor \((x-3)\) gives root \(x = +3\). Sign reversal is the single most common error here.
Answer: A
QUESTION 9
Solve \(x^2 + 5x + 6 = 0\).
(A) \(x = -2\) or \(x = -3\)(B) \(x = 2\) or \(x = 3\) (C) \(x = -1\) or \(x = -6\)(D) \(x = 6\) or \(x = 1\)
Solution: Two numbers: product \(+6\), sum \(+5\) → \(2\) and \(3\).
\((x + 2)(x + 3) = 0 \Rightarrow x = -2\) or \(x = -3\).
Factor \((x+2)\) gives root \(x = -2\). Both signs are negative here because both factor constants are positive.
Answer: A
QUESTION 10
Solve \(x^2 - x - 6 = 0\).
(A) \(x = 3\) or \(x = -2\)(B) \(x = -3\) or \(x = 2\) (C) \(x = 3\) or \(x = 2\)(D) \(x = -3\) or \(x = -2\)
Solution: Product \(-6\), sum \(-1\): factors \(-3\) and \(+2\).
\((x - 3)(x + 2) = 0 \Rightarrow x = 3\) or \(x = -2\).
One root is positive, one is negative — the mixed signs come from the negative product.
Answer: A
QUESTION 11
Solve \(2x^2 + 5x - 3 = 0\) by factorisation.
(A) \(x = \frac{1}{2}\) or \(x = -3\)(B) \(x = \frac{1}{2}\) or \(x = 3\) (C) \(x = -\frac{1}{2}\) or \(x = 3\)(D) \(x = 2\) or \(x = -\frac{3}{2}\)
Solution: AC method: \(a \cdot c = 2 \times (-3) = -6\). Need factors of \(-6\) summing to \(5\): \(6\) and \(-1\).
\(2x^2 + 6x - x - 3 = 2x(x+3) - 1(x+3) = (2x-1)(x+3) = 0\).
\(x = \frac{1}{2}\) or \(x = -3\).
Answer: A
QUESTION 12
Solve \(x^2 - 9 = 0\).
(A) \(x = 3\) or \(x = -3\)(B) \(x = 9\) (C) \(x = 3\) only(D) \(x = \sqrt{9}\) only
Solution: \(x^2 = 9 \Rightarrow x = \pm 3\). Alternatively: difference of two squares \((x-3)(x+3) = 0\).
A squared equation always has two solutions (unless both coincide). Never forget the negative root.
Answer: A
QUESTION 13
The roots of \(x^2 + bx + 10 = 0\) are \(x = -2\) and \(x = -5\). Find \(b\).
(A) 7(B) −7(C) 10(D) 3
Solution: Sum of roots \(= -2 + (-5) = -7 = -b\), so \(b = 7\). Verify product: \((-2)(-5) = 10\) ✓.
Sum of roots \(= -b/a\). With \(a = 1\), sum \(= -b\).
Answer: A
QUESTION 14
Solve \(3x^2 - 12 = 0\).
(A) \(x = 2\) or \(x = -2\)(B) \(x = 4\) (C) \(x = 2\) only(D) \(x = \sqrt{12}\)
Solution: Divide by 3: \(x^2 - 4 = 0 \Rightarrow (x-2)(x+2) = 0 \Rightarrow x = \pm 2\).
Always divide by the leading coefficient first when there is no \(x\)-term. Both roots are valid.
Answer: A
🔹 Topic 3: The Quadratic Formula — \(x = \dfrac{-b \pm \sqrt{b^2 - 4ac}}{2a}\) (Direct Q15–Q21)
Core trap 1: the formula uses −b, so if b = −5, then −b = +5. Core trap 2: the entire numerator (−b ± √·) is divided by 2a, not just the ±√· part.
QUESTION 15
Use the quadratic formula to solve \(x^2 - 5x + 6 = 0\).
(A) \(x = 3\) or \(x = 2\)(B) \(x = -3\) or \(x = -2\) (C) \(x = 5\) or \(x = 1\)(D) \(x = 6\) or \(x = 1\)
Solution: \(a=1, b=-5, c=6\). Discriminant: \(b^2 - 4ac = 25 - 24 = 1\).
\(x = \frac{-(-5) \pm \sqrt{1}}{2(1)} = \frac{5 \pm 1}{2}\). So \(x = 3\) or \(x = 2\).
Since \(b = -5\), then \(-b = +5\). Students who write \(-(-5) = -5\) will get the wrong answer.
Answer: A
QUESTION 16
Solve \(2x^2 - 3x - 2 = 0\) using the quadratic formula.
(A) \(x = 2\) or \(x = -\frac{1}{2}\)(B) \(x = -2\) or \(x = \frac{1}{2}\) (C) \(x = 3\) or \(x = -1\)(D) \(x = 2\) or \(x = 1\)
Solution: \(a=2,b=-3,c=-2\). Discriminant: \(9+16 = 25\).
\(x = \frac{3 \pm 5}{4}\). So \(x = \frac{8}{4} = 2\) or \(x = \frac{-2}{4} = -\frac{1}{2}\).
The full numerator is divided by \(2a = 4\). Do not divide only the \(\pm 5\) part.
Answer: A

Solution: \(\Delta = 4^2 - 4(1)(4) = 16 - 16 = 0\). One repeated root: \(x = -2\).
When \(\Delta = 0\) the parabola touches the x-axis at exactly one point (vertex on axis).

Solution: \(\Delta = 2^2 - 4(3)(5) = 4 - 60 = -56 < 0\). No real roots.
A negative discriminant means the parabola does not cross the x-axis at all.

Solve \(x^2 - 6x + 5 = 0\) using the formula. Give both roots.

Solve \(x^2 - 2x - 1 = 0\), leaving the answer in surd form.

🔹 Topic 4: Sum & Product of Roots — Vieta's Formulas (Direct Q22–Q28)
For ax² + bx + c = 0 with roots α and β: Sum α + β = −b/a. Product αβ = c/a. The MINUS sign in the sum formula is the number-one mistake.
QUESTION 22
The roots of \(x^2 - 7x + 10 = 0\) are \(\alpha\) and \(\beta\). Find \(\alpha + \beta\).
(A) 7(B) −7(C) 10(D) 3
Solution: \(\alpha + \beta = \frac{-b}{a} = \frac{-(-7)}{1} = 7\).
With \(b = -7\), the formula gives \(-(-7)/1 = +7\). Students who forget the sign get \(-7\).
Answer: A
QUESTION 23
For the equation \(x^2 - 7x + 10 = 0\), find \(\alpha\beta\).
(A) 10(B) −10(C) 7(D) −7
Solution: \(\alpha\beta = \frac{c}{a} = \frac{10}{1} = 10\).
Product formula carries no sign change — it is simply \(c/a\).
Answer: A
QUESTION 24
The roots of \(2x^2 + 3x - 5 = 0\) are \(\alpha\) and \(\beta\). Find \(\alpha + \beta\).
(A) \(-\frac{3}{2}\)(B) \(\frac{3}{2}\) (C) \(-\frac{5}{2}\)(D) \(5\)
Solution: \(\alpha + \beta = \frac{-3}{2}\).
Here \(a = 2\), \(b = 3\), so sum \(= -3/2\). Division by \(a\) is essential when \(a \neq 1\).
Answer: A

For the same equation \(2x^2 + 3x - 5 = 0\), find \(\alpha\beta\).
(A) \(-\frac{5}{2}\)(B) \(\frac{5}{2}\) (C) \(-5\)(D) \(\frac{3}{2}\)
Solution: \(\alpha\beta = \frac{c}{a} = \frac{-5}{2}\).
Answer: A
QUESTION 26
Given \(\alpha + \beta = 5\) and \(\alpha\beta = 6\), write the quadratic equation with roots \(\alpha\) and \(\beta\).
(A) \(x^2 - 5x + 6 = 0\)(B) \(x^2 + 5x + 6 = 0\) (C) \(x^2 - 5x - 6 = 0\)(D) \(x^2 + 6x - 5 = 0\)
Solution: Equation: \(x^2 - (\alpha+\beta)x + \alpha\beta = 0 \Rightarrow x^2 - 5x + 6 = 0\).
The sum becomes the coefficient of \(x\) with a MINUS sign. If you write \(+5x\), you have the sum of roots as \(-5\), not \(+5\).
Answer: A
QUESTION 27
The roots of \(x^2 - 5x + 6 = 0\) are \(\alpha\) and \(\beta\). Find \(\alpha^2 + \beta^2\).
(A) 13(B) 25(C) 1(D) 37
Solution: Use identity: \(\alpha^2 + \beta^2 = (\alpha+\beta)^2 - 2\alpha\beta = 5^2 - 2(6) = 25 - 12 = 13\).
Do NOT try to find \(\alpha\) and \(\beta\) separately. Use the identity — it is faster and reinforces Vieta's use.
Answer: A
QUESTION 28
Given roots \(\alpha = 3\) and \(\beta = -\frac{1}{2}\), form the quadratic equation with integer coefficients.
(A) \(2x^2 - 5x - 3 = 0\)(B) \(2x^2 + 5x - 3 = 0\) (C) \(x^2 - \frac{5}{2}x - \frac{3}{2} = 0\)(D) \(2x^2 - 3x - 5 = 0\)

When roots are fractions, multiply the whole equation by the LCM of denominators to clear fractions.

🔹 Topic 6: Zeros, Roots & the Factor Theorem (Direct Q36–Q42)
Vocabulary precision: the ZERO of f(x) is the x-value where f(x)=0. The ROOT of f(x)=0 is the same value. If x = r is a root, then (x − r) is a FACTOR. These three things always travel together.
QUESTION 36
Which of the following is a zero of \(f(x) = x^3 - 6x^2 + 11x - 6\)?
(A) \(x = 1\)(B) \(x = -1\) (C) \(x = 6\)(D) \(x = 0\)
Solution: Test \(x=1\): \(1 - 6 + 11 - 6 = 0\) ✓. So \(x=1\) is a zero.
If \(f(1) = 0\), then \(x = 1\) is a zero AND \((x-1)\) is a factor.
Answer: A
QUESTION 37
If \(f(x) = x^3 + 2x^2 - 5x - 6\) and \(f(2) = 0\), which is a factor of \(f(x)\)?
(A) \((x - 2)\)(B) \((x + 2)\) (C) \((2x - 1)\)(D) \((x - 1)\)
Solution: Factor theorem: if \(f(r) = 0\), then \((x - r)\) is a factor. Since \(f(2)=0\), factor is \((x-2)\).
Students who see \(f(2)=0\) and write factor \((x+2)\) are applying the wrong sign. The root is +2 so the factor is \((x-2)\).
Answer: A
QUESTION 38
Verify that \(x = -1\) is a zero of \(f(x) = x^3 + 2x^2 - 5x - 6\), then fully factorise \(f(x)\).
(A) \((x+1)(x-2)(x+3)\)(B) \((x-1)(x+2)(x-3)\) (C) \((x+1)(x+2)(x-3)\)(D) \((x+1)^2(x-6)\)
Solution: \(f(-1) = -1+2+5-6 = 0\) ✓. Divide by \((x+1)\): quotient is \(x^2+x-6=(x-2)(x+3)\).
So \(f(x) = (x+1)(x-2)(x+3)\).
Answer: A
QUESTION 39
The zeros of \(f(x) = x^3 + 2x^2 - 5x - 6\) are:
(A) \(-1, 2, -3\)(B) \(1, -2, 3\) (C) \(-1, -2, -3\)(D) \(1, 2, 3\)
Solution: From Q38: factors are \((x+1)(x-2)(x+3)\). Setting each to zero: \(x = -1, 2, -3\).
Read zeros from factors: factor \((x+1) \Rightarrow x = -1\). Factor \((x-2) \Rightarrow x = 2\). Factor \((x+3) \Rightarrow x = -3\).

🔹 Topic 7: Inverse Functions (Direct Q43–Q49)
Critical distinction: f⁻¹(x) means the inverse function — the rule that UNDOES f. It is NOT the reciprocal 1/f(x). The method: write y = f(x), swap x and y, then make y the subject.
QUESTION 43
Find the inverse of \(f(x) = 2x + 3\).
(A) \(f^{-1}(x) = \dfrac{x - 3}{2}\)(B) \(f^{-1}(x) = \dfrac{x + 3}{2}\) (C) \(f^{-1}(x) = \dfrac{1}{2x+3}\)(D) \(f^{-1}(x) = 2x - 3\)
Solution: Let \(y = 2x + 3\). Swap: \(x = 2y + 3\). Solve for \(y\): \(y = \frac{x-3}{2}\).
So \(f^{-1}(x) = \frac{x-3}{2}\).
Option C is the reciprocal — that is \([f(x)]^{-1}\), NOT \(f^{-1}(x)\). They are different objects.
Answer: A
QUESTION 44
Find \(f^{-1}(x)\) for \(f(x) = \dfrac{x + 1}{3}\).
(A) \(f^{-1}(x) = 3x - 1\)(B) \(f^{-1}(x) = 3(x+1)\) (C) \(f^{-1}(x) = \dfrac{3}{x+1}\)(D) \(f^{-1}(x) = \dfrac{1}{3}x - 1\)
Solution: \(y = \frac{x+1}{3}\). Swap: \(x = \frac{y+1}{3}\). Multiply by 3: \(3x = y+1\). So \(y = 3x - 1\).
Answer: A
QUESTION 45
Given \(f(x) = 3x - 1\), find \(f^{-1}(5)\).
(A) 2(B) 14(C) \(\frac{1}{14}\)(D) \(\frac{5}{3}\)
Solution: First find \(f^{-1}(x)\): \(y = 3x-1 \Rightarrow x = 3y-1 \Rightarrow y = \frac{x+1}{3}\).
Then \(f^{-1}(5) = \frac{5+1}{3} = \frac{6}{3} = 2\).
Alternatively: solve \(3x - 1 = 5 \Rightarrow x = 2\). The input that produces output 5 is 2.
Answer: A
QUESTION 46
Verify that \(f(f^{-1}(x)) = x\) for \(f(x) = 2x + 3\) and \(f^{-1}(x) = \frac{x-3}{2}\).
(A) \(f(f^{-1}(x)) = x\) ✓ — confirms inverse is correct (B) \(f(f^{-1}(x)) = 2x\) (C) \(f(f^{-1}(x)) = x + 3\) (D) \(f(f^{-1}(x)) = \frac{x}{2}\)
Solution: \(f\!\left(\frac{x-3}{2}\right) = 2 \cdot \frac{x-3}{2} + 3 = (x-3)+3 = x\) ✓.
If \(f(f^{-1}(x)) = x\), the inverse is correct. This is the key verification step.
