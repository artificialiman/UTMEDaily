● Live IT Attachment Community Tool
Surulere Digital Health Board
Chika Eze · UNILAG Computer Science · IT 2025
A low-bandwidth community health dashboard for the Surulere LGA clinic network. Built as an IT attachment project. Displays real-time bed availability and public health alerts on a TV screen at 3 clinics.
Live Demo GitHub Walkthrough
02
In Progress Final Year
Abuja Pothole Reporter
Fatima Al-Hassan · ABU Zaria · Engineering · Class of 2026
Crowdsourced pothole mapping tool for FCT. Citizens photograph, GPS-tag, and submit. Data feeds to FCT Works Ministry portal. Final-year project doubling as civic infrastructure.
Prototype Report
03
● Live NYSC Project
Kano Market Price Tracker
Umar Suleiman · Corp Member · Kano State · Batch A 2025
WhatsApp-first price transparency tool for Kano's Singer Market. Traders submit daily prices via WhatsApp; app aggregates and broadcasts weekly summaries to 1,200 subscribers.
